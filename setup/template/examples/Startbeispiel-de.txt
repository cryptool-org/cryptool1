CrypTool (Startbeispiel zur CrypTool-Versionsfamilie 1.x)

CrypTool ist ein umfangreiches freies Lernprogramm zu den Themen Kryptographie und Kryptoanalyse
mit ausführlicher Online-Hilfe und mit vielen Visualisierungen.

Dies ist eine Textdatei, mit der Sie Ihre ersten Schritte mit CrypTool machen können.

1) Den besten Überblick über die Möglichkeiten von CrypTool bietet die "Startseite", die sich in der Windows Online-Hilfe befindet. Von der Startseite aus können Sie alle wesentlichen Funktionen über Links erreichen.
Die Startseite selbst erreichen Sie über das Menü "Hilfe -> Startseite" oder indem Sie in der Online-Hilfe im Index den Begriff "Startseite" eingeben.
Sie können die Online-Hilfe mit F1 an jeder Stelle in CrypTool öffnen.

2) Als nächstes könnten Sie eine Datei z.B. mit dem Caesar-Verfahren verschlüsseln -- über das Menü "Ver-/Entschlüsseln -> Symmetrisch (klassisch)".

3) Die Online-Hilfe enthält verschiedene Szenarien (Tutorials), die einen schnellen Einstieg in die Verfahren der Kryptologie ermöglichen. Die Szenarien finden Sie über das Menü "Hilfe -> Szenarien (Tutorials)".

4) Weitere Funktionen können Sie z.B. auf folgende Weise erkunden:
- Sie klicken sich spielerisch durch die Menüs. Dabei können Sie zu jedem ausgewählten Menüeintrag Informationen erhalten, indem Sie F1 drücken.
- Sie schauen sich die Readme-Datei an (siehe Menü "Hilfe -> Readme").
- Sie schauen sich die beiliegende bunte Präsentation an (Diese Präsentation ist auf verschiedene Arten zugänglich: z.B. unter dem Menü "Hilfe", oder über den Abschnitt "Dokumentation" auf der Startseite der Online-Hilfe).
- Sie schauen sich die Webseite www.cryptool.de an.
