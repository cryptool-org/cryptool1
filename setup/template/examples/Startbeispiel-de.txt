CrypTool

CrypTool ist ein umfangreiches Lernprogramm zu den Themen Kryptographie und Kryptoanalyse.

Dies ist eine Textdatei, mit der Sie Ihre ersten Schritte mit CrypTool machen können.

1) Sie können diese Datei z.B. über das Menü "Ver-/Entschlüsseln \ Symmetrisch (klassisch)" mit dem Caesar-Verfahren verschlüsseln.

2) Den besten Überblick über die Möglichkeiten von CrypTool bietet die Startseite in der Windows Online-Hilfe zu CrypTool. Von der Startseite aus können Sie alle wesentlichen Funktionen über Links erreichen.
Die Startseite selbst erreichen Sie über das Menü "Hilfe \ Startseite" oder indem Sie in der Online-Hilfe im Index den Begriff "Startseite" eingeben.
Öffnen Sie die Online-Hilfe mit F1.

3) Insbesondere mit den Szenarien (Tutorials) in der Online-Hilfe finden Sie einen schnellen Einstieg. Auch die Szenarien können Sie direkt über das Menü "Hilfe" erreichen.

4) Weitere Features können Sie z.B. auf folgende Weise erkunden:
- Sie klicken sich spielerisch durch die Menüs. Mit F1 erhalten Sie die Hilfe zu dem ausgewählten Menüeintrag.
- Sie schauen sich unter dem "Hilfe"-Menü die Readme-Datei an.
- Sie schauen sich die beiliegende bunte Präsentation an (sie ist ebenfalls unter dem "Hilfe"-Menü zu finden; oder über den Abschnitt "Dokumentation" auf der Startseite der Online-Hilfe).
- Sie schauen sich die Webseite www.cryptool.de an.
