CrypTool

This is a text file, in order to help you to make your first steps with CrypTool.

One thing you can do e.g. is to encrypt this file with the Caesar algorithm (via the menu Crypt \ Classical).

Using the search keyword "Starting page" within the index of the Windows online help, you will see what else you can do with CrypTool.
From the starting page you can get to all relevant functions via links.

Especially with the examples (tutorials) provided with the online help makes it easy for you to get up to speed.
