Dialog Öffentliche Parameter setzen

Dieser Dialog erscheint, wenn Sie aus der Diffie-Hellman-Demo heraus die Schaltfläche Öffentliche Parameter setzen klicken.

Der Dialog fordert Sie auf, die öffentlichen Parameter für die am Schlüsselaustausch beteiligten Parteien festzulegen.

Geben Sie bitte zunächst den Primzahlmodul p ein, da die Verifikation des Generators g direkt von p abhängig ist.

Beachten Sie, dass die Verwendung einer selbstgewählten Zahl, insbesondere wenn diese nicht prim ist, die Sicherheit des Verfahrens negativ beeinflussen kann!

Über die Schaltfläche Primzahl erzeugen können Sie sich eine sogenannte sichere Primzahl erzeugen. In der dadurch aufgerufenen Dialogbox Primzahlen generieren müssen Sie nach dem Generieren auf die Schaltfläche Primzahlen übernehmen klicken.

Beachten Sie bitte, dass die Größe bzw. Bitlänge des Primzahlmoduls p ein wesentlicher Aspekt im Bezug auf die Sicherheit des Verfahrens ist. Für Demonstrationszwecke können Sie natürlich auch Werte wählen, die in der Realität als unsicher gelten.

Analog zur Auswahl des Primzahlmoduls p können Sie auch den Wert für den Generator g vom Dialog wählen lassen; betätigen Sie dazu die Schaltfläche Generator erzeugen. Falls Sie zuvor eine sichere Primzahl erzeugt haben, kann ein Generator oder auch eine Primitivwurzel in der multiplikativen Gruppe modulo p bestimmt werden. Falls Sie selbst einen Generator auswählen wollen, dann kann CrypTool nicht für die Generatoreigenschaft garantieren. Bitte beachten Sie, dass Sie für g keine der folgenden Werte

  • Null
  • Eins
  • Primzahlmodul – 1
  • Vielfaches des Primzahlmoduls

zugeweisen, da das Verfahren ansonsten nicht mehr sicher ist.

Mit der Schaltfläche Parameter übernehmen schließen Sie den Dialog und gelangen zum nächsten Schritt des Diffie-Hellman-Verfahrens.
