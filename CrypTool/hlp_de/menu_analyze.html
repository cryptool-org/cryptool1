Befehle des Menüs Analyse

Das Menü Analyse enthält folgende Befehle:

Allgemein Allgemeine statistische Informationen über das aktive Dokument.
Zufalltests Analyse der Zufallseigenschaften des aktiven Dokuments.
Algorithmen (automatische Analyse) Liste der Algorithmen, für die eine automatische Analyse implementiert ist.
Ciphertext only Automatische Analyse, wobei nur der verschlüsselte Text gegeben ist.
Known plaintext Automatische Analyse, wobei Klartext und Chiffrat gegeben sind.
Manuelle Analyse Unterstützung für Benutzer-gesteuerte Analysen.

Die Menü-Einträge Allgemein, Ciphertext only und Algorithmen stehen sowohl für Hex- als auch als auch für Text-Dokumente zur Verfügung.

Die Menü-Einträge Known plaintext und Manuelle Analyse stehen nur für Text-Dokumente zur Verfügung.

Der Menü-Eintrag Zufalltests steht nur für Binärdateien zur Verfügung. Zufalltests liefern auch eine Art allgemeine Information zum aktiven Dokument.

Im folgenden werden die einzelnen Einträge der Untermenüs und die Struktur der Angriffe erläutert.

  1. Allgemeine Informationen über das aktive Dokument:
    2. Entropie Berechnet die Entropie des Dokuments.
    Gleitende Häufigkeit Berechnet die gleitende Häufigkeit des Dokuments.
    Histogramm Berechnet die Häufigkeit der in dem Dokument auftretenden Buchstaben.
    N-Gramm Analysiert die Häufigkeit von N-Grammen.
    Autokorrelation Berechnet die Autokorrrelation des Dokuments.
    Vitany Gibt ein Vitany-Diagramm aus.
    Periode Analysiert die Periodizität.

    Der Menüpunkt Vitany steht nur für Binärdateien zur Verfügung.

  2. Automatische Analysen:
    1. Algorithmen:
      2. Algorithmen Hier werden alle in b) und c) vorhandenen Verfahren aufgelistet, für die eine automatische Analyse implementiert ist.

      In b) und c) werden die beiden Angriffsformen Ciphertext only und Known plaintext unterschieden (siehe dazu das Kapitel Angriffsszenarien):

    2. "Ciphertext-only" Angriffe:
      3. Caesar Automatische Analyse für das Caesar-Verschlüsselungsverfahren.
      Vigenère Automatische Analyse für das Vigenère-Verschlüsselungsverfahren
      Addition Automatische Analyse für byteweise Addition.
      XOR Automatische Analyse für binäres exklusives Oder.
      IDEA Automatische Analyse für IDEA-Verschlüsselungsverfahren.
      RC2 Automatische Analyse für RC2-Verschlüsselungsverfahren.
      RC4 Automatische Analyse für RC4-Verschlüsselungsverfahren.
      DES (ECB) Automatische Analyse für DES-Verschlüsselungsverfahren im ECB-Modus.
      DES (CBC) Automatische Analyse für DES-Verschlüsselungsverfahren im CBC-Modus.
      Triple DES (ECB) Automatische Analyse für Triple DES-Verschlüsselungsverfahren im ECB-Modus.
      Triple DES (CBC) Automatische Analyse für Triple DES-Verschlüsselungsverfahren im CBC-Modus.
      MARS Automatische Analyse für MARS-Verschlüsselungsverfahren.
      RC6 Automatische Analyse für RC6-Verschlüsselungsverfahren.
      Rijndael Automatische Analyse für Rijndael-Verschlüsselungsverfahren.
      Serpent Automatische Analyse für Serpent-Verschlüsselungsverfahren.
      Twofish Automatische Analyse für Twofish-Verschlüsselungsverfahren.

      Die Analyseverfahren für Caesar und Vigenère können nur auf Textdateien, die restlichen Verfahren nur auf Binärdateien angewendet werden.

    3. "Known-plaintext" Angriffe:
      4. Hill Automatische Analyse für das Hill-Verschlüsselungsverfahren. Es kann nur auf Textdateien angewandt werden.

  3. Manuelle Analysen:
    4. Substitution Automatische Analyse für monoalphabetische Substitution.
    Playfair Analysehilfe für den Playfair-Algorithmus.

    Diese beiden Analyseverfahren können nur auf Textdateien angewandt werden.

  4. Algorithmen zu Zufallstests:
    5. Frequency-Test Führt den Frequency-Test an dem Dokument durch.
    Poker-Test Führt den Poker-Test an dem Dokument durch.
    Run-Test Führt den Run-Test an dem Dokument durch.
    Serial-Test Führt den Serial-Test an dem Dokument durch.
    FIPS-PUB-140-1 Test-Batterie Führt die FIPS-PUB-140-1 Test-Batterie an dem Dokument durch.

    Die Zufallstests stehen nur für Binärdateien zur Verfügung.
