Datei

Eine Datei enthält ein Dokument, das in CrypTool geöffnet werden kann. Informationen zum Erstellen und Bearbeiten von Dokumenten finden Sie im Hilfeindex.

Eine Datei besitzt einen Namen, ihren Dateinamen. So kann beispielsweise ein verschlüsseltes Dokument in einer Datei unter einem Namen gespeichert werden.
