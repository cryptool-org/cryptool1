Dialog Dokument-Eigenschaften

Diesen Dialog erreichen Sie über den Befehl Dokument-Eigenschaften im Menü Datei.

Das Dialogfenster Dokument-Eigenschaften zeigt u.a., wo das im CrypTool-Hauptfenster angezeigte Dokument (initial) gespeichert wird, seinen Typ und seine Größe.

Da jedes angezeigte Dokument geändert werden kann, wird dazu immer auch eine temporäre Datei geschrieben. Die Namen dieser Dateien beginnen standardmäßig mit "CRY".
