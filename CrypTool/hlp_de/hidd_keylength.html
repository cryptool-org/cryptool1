Dialog Automatische Vigenère-Analyse (ermittelte Schlüssellänge ausgeben)

Sie erreichen diesen Dialog über den Menüeintrag Analyse \ Symmetrische Verschlüsselung(klassisch) \ Ciphertext only \ Vigenère.

Dieser Dialog gehört zur automatischen Analyse des Vigenère-Verschlüsselungsverfahrens.

In diesem Dialog wird die ermittelte Schlüssellänge ausgegeben. Sofern die Länge falsch ist, kann sie geändert werden.

Die Betätigung des Weiter-Knopfes führt zu einer Fortsetzung der Analyse, Abbrechen beendet die Analyse.
