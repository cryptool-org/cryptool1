Dialog Optionen - Automatische Analyse monoalphabetische Substitution basierend auf Digrammen (Methode 1)

Sie erreichen diesen Dialog, nachdem Sie Methode 1 auswählten im Dialog Methodenauswahl zur automatischen Substitutionsanalyse.

Im Dialog haben Sie zwei Checkboxen zur Auswahl,

  1. ob das Leerzeichen (Blank) ein Teil des Alphabets ist
  2. ob Zwischenergebnisse angezeigt werden sollen.
