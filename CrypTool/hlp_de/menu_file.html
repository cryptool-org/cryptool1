Befehle des Menüs Datei

Das Menü Datei enthält die folgenden Befehle:

Neu Erstellt ein neues Dokument.
Öffnen Öffnet ein bestehendes Dokument.
Schließen Schließt ein geöffnetes Dokument.
Speichern Speichert ein geöffnetes Dokument unter seinem Dateinamen.
Speichern als Speichert ein geöffnetes Dokument unter einem angegebenen Dateinamen.
Drucken Druckt ein Dokument.
Drucker einstellen Wählt einen Drucker und eine Druckerverbindung.
1,2,3,... Öffnet eines der zuletzt geschlossenen Dokumente.
Beenden Beendet CrypTool.
