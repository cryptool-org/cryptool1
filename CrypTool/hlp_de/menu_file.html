Befehle des Top-Level-Menüs Datei

Das Menü Datei enthält die folgenden Befehle:

Neu Erstellt ein neues Dokument.
Öffnen Öffnet ein bestehendes Dokument.
Schließen Schließt das aktive Dokument.
Speichern Speichert das aktive Dokument.
Speichern als Speichert das aktive Dokument unter einem neuen Namen.
Dokument-Eigenschaften Zeigt die Eigenschaften des aktiven Dokuments an.
Drucken Druckt das aktive Dokument.
Drucker einstellen Ändert den Drucker bzw. die Druckereinstellungen.
Kürzliche Dateien / 1,2,3,... Liste zum Öffnen der zuletzt gespeicherten Dokumente.
Beenden Beendet die Anwendung mit Nachfrage, ob die geöffneten Dateien gespeichert werden sollen.
