Der Befehl "Speichern als" (Menü Datei)

Verwenden Sie diesen Befehl, um das aktive Dokument zu speichern und zu benennen. CrypTool zeigt den Dialog Speichern als an, so dass Sie einen Namen für Ihr Dokument angeben können.

Zur Speicherung eines Dokuments unter seinem bestehenden Namen und Verzeichnis dient der Befehl Speichern.
