Hilfe zur CrypTool-Hilfe

Für CrypTool gibt es eine Online-Hilfe im typischen Windows-Stil (HTML Help). Die Bedienung geschieht durch Anklicken von hervorgehobenen (unterstrichenen und andersfarbig unterlegten) Begriffen in den Hilfetexten (Hyperlinks).

Dabei wird durch Anklicken eines mit einer durchgezogenen Linie unterstrichenen Themas dieses neue Thema im Hilfe-Fenster angezeigt. Zum vorherigen Thema gelangen Sie über den Zurück-Knopf der Hilfe.

Ist ein Thema mit einer gestrichelten Linie unterstrichen, so bleibt das aktuelle Thema im Fenster erhalten und es wird nur ein kleines Fenster mit einer kurzen Erklärung gezeigt.

Außerdem kann über die Knöpfe Inhalt und Index nach Themen oder Begriffen gesucht werden.

Es gibt eine Hilfe über Hilfe, das heißt eine Hilfe zur Hilfe (HTML Help), die genauso aufgebaut ist wie die Hilfe zu CrypTool. Diese kann mit der Taste F1 aufgerufen werden, wenn Sie – wie im Moment – die Hilfe zu CrypTool geöffnet haben und diese das aktive Fenster ist.

Die wesentlichen Eigenschaften der Online-Hilfe werden im folgenden genauer erläutert

Über das Menü können unter anderem einzelne Hilfeseiten gedruckt werden und verschiedene Optionen eingestellt werden.

Die Online-Hilfe verfügt über verschiedene Knöpfe. Die Knöpfe Inhalt und Index öffnen einen Dialog, in dem man über die verschiedenen Registerkarten auf unterschiedliche Weise Hilfe zu den gewünschten Themen erhalten kann.

Die Online-Hilfe enthält zu jedem Dialog und jedem Menüpunkt sowie den Symbolen der Symbolleiste eine Hilfeseite.

Außerdem können verschiedene Tutorials / Szenarien / Beispiele (erreichbar über den Hilfeindex, Szenarien und das Skript) abgerufen werden, so zum Beispiel über Verschlüsselungsverfahren und über Primzahlen.

Öffnen der Online-Hilfe

Die Hilfe zum Programm CrypTool kann auf verschiedene Arten angefordert werden:

  • Bei jedem Start von CrypTool, wenn man F1 drückt. Auf dieser Startseite erhält man Informationen über den Beginn der Arbeit mit CrypTool und die in CrypTool enthaltenen Funktionen.
  • Durch Auswahl des Befehls Inhalt aus dem Menü Hilfe wird der Hilfe-Index geöffnet. Durch Doppelklicken der Einträge können verschiedene Hilfethemen geöffnet werden. Ferner kann über die Registerkarte Index nach bestimmten Begriffen gesucht werden. Eine Volltextsuche steht mit der Registerkarte Suchen zur Verfügung.
  • Durch Drücken der Taste F1 wird die kontextabhängige Hilfe gestartet. So erhält man zu dem gerade aktiven Fenster oder Menüpunkt die entsprechende Hilfeseite.
  • Durch Drücken der Tastenkombination Umschalt+F1 oder durch Klicken auf das Symbol Hilfe (das ist das Symbol mit einem Mauszeiger und einem Fragezeichen) in der Symbolleiste verändert sich der Mauszeiger (es erscheint zusätzlich ein Fragezeichen); ein anschließendes Klicken auf ein Element des Fensters von CrypTool ruft die Hilfeseite zu dem entsprechenden Element auf.
  • Außerhalb der Anwendung: Durch Aufrufen der CrypTool Hilfe-Datei, die sich im gleichen Ordner wie CrypTool befindet.

Schließen der Online-Hilfe

Die Hilfe kann über das Menü Datei \ Beenden, die Schaltfläche Beenden in der Titelleiste oder einfacher durch Betätigen der Esc-Taste beendet werden.

Wenn Sie CrypTool beenden, wird die geöffnete Hilfe ebenfalls beendet.
