Dialog Schlüssel-Details

Über diesen Dialog erhalten Sie zusätzliche Informationen zum Session Key, auf den sich Alice und Bob geeinigt haben.

Neben der Schlüssellänge für jeden der beiden Schlüssel von Alice und Bob wird ausgegeben, ob die beiden Schlüssel überhaupt identisch sind.

Ist die Verifikation erfolgreich, so können Alice und Bob nun theoretisch den Session Key zum Verschlüsseln von Nachrichten untereinander verwenden.
