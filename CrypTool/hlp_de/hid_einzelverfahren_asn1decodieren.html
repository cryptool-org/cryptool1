ASN.1-Decodieren einer Datei (Menü Einzelverfahren \ Tools \ Codierungen)

Durch die Auswahl dieser Menüoption lässt sich das aktuell ausgewählte Dokument – wenn möglich – mit Hilfe des Dialogs ASN.1-PIN-Eingabe ASN.1-decodieren.

Handelt es sich tatsächlich um ein ASN.1-codiertes Dokument, so wird die ASN.1-Struktur in Form eines ASN.1-Dumps in einem neuen Fenster ausgegeben.

Beinhaltet das gewählte ASN.1-Dokument zudem eine PSE, in der sich ein Zertifikat befindet, so wird dieses Zertifikat in einem weiteren Fenster in textueller Form angezeigt.

Findet CrypTool innerhalb der ausgewählten Datei keine bekannten ASN.1-Strukturen, so erscheint ein Dialog mit den entsprechenden Informationen.

Bemerkung:

Zu beachten ist hierbei, dass die gewählte PSE durchaus PIN-geschützt sein kann. In diesem Fall werden Sie durch einen Dialog zur Eingabe einer PIN aufgefordert, damit ein unautorisierter Zugriff auf das Zertifikat nicht möglich ist.
