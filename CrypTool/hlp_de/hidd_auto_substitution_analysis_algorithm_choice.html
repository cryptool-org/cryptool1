Dialog Methodenauswahl zur automatischen Substitutionsanalyse

Sie erreichen diesen Dialog über den Menüeintrag Analyse \ Symmetrisch Verschlüsselung (klassisch) \ Ciphertext only \ Substitution.

Hier können Sie aus zwei Möglichkeiten wählen, die automatische Substitutionsanalyse durchzuführen:

  • Methode 1 basierend auf der Häufigkeitsanalyse von Digrammen,
  • Methode 2 basierend auf den häufigsten Wörtern.
Die Standard-Einstellung ist deutscher Klartext.
