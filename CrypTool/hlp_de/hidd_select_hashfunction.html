Dialog Hashfunktion auswählen

Der Hashwert oder Fingerabdruck einer Datei ist abhängig von dem gewählten Hashverfahren. In diesem Dialog wählen Sie das Hashverfahren aus (Auswahl durch die Radiobutton-Liste):

MD2 128 Bit
MD4 128 Bit
MD5 160 Bit
SHA 128 Bit
SHA-1 160 Bit
RIPEMD-160 160 Bit

Beachten Sie, dass die PKCS #1-Spezifikation für RSA-Signaturen das MD4-Hashverfahren nicht vorsieht.
