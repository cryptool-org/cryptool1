Dialog Hashfunktion wählen

Der Hashwert oder Fingerabdruck einer Datei ist abhängig von dem gewählten Hashverfahren. In diesem Dialog wählen Sie das Hashverfahren (in der Radiobutton-Liste) aus:

MD2 128 Bit
MD4 128 Bit
MD5 160 Bit
SHA 128 Bit
SHA-1 160 Bit
SHA-256 256 Bit
SHA-512 512 Bit
RIPEMD-160 160 Bit

Beachten Sie, dass die offizielle PKCS #1-Spezifikation für RSA-Signaturen (RFC 3447) folgende Hashverfahren nicht vorsieht: MD4, SHA, RIPEMD-160.
