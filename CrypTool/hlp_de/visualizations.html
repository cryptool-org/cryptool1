Visualisierungen in CrypTool

Visualisierungen (interaktive Demonstrationen) finden Sie zu den Themen:

  1. Erzeugung einer elektronischen Signatur,

  2. Hybridverschlüsselung,

  3. Wirkung von Textänderungen auf den Hashwert

    1. Sensitivität von Hashverfahren,

    2. Erzeugung von Hashwert-Kollisionen (Anwendung des Geburtstagsparadoxons),

  4. Ablauf des Diffie-Hellman-Schlüsselaustauschverfahrens,

  5. Seitenkanalangriff,

  6. Authentisierungsverfahren,

  7. Shared Secret

    1. basierend auf dem Chinesischen Restsatz,

    2. nach Shamir,

  8. 3D-Visualisierung von großen Mengen von Zufallszahlen.

Diese erreichen Sie über die Menüs:

  1. Einzelverfahren \ RSA-Demo \ Signaturdemo (Signaturerzeugung)

  2. Ver-/Entschlüsseln \ Hybrid-Demo \ Hybridverschlüsselung
    (danach kann für ein Fenster, das das Ergebnis der Hybridverschlüsselung enthält, die Hybrid-Entschlüsselung aufgerufen werden.)

  3. Wirkung von Textänderungen auf den Hashwert

    1. Einzelverfahren \ Hashwerte \ Hash-Demo

    2. Einzelverfahren \ Angriff auf den Hashwert der digitalen Signatur

  4. Einzelverfahren \ Diffie-Hellman-Demo

  5. Analyse \ Asymmetrische Verfahren \ Seitenkanalangriff auf "Textbook-RSA"

  6. Einzelverfahren \ Protokolle \ Authentisierungsverfahren im Netz

  7. Shared Secret

    1. Einzelverfahren \ Secret-Sharing-Demo (nach Shamir)

    2. Einzelverfahren \ Anwendungen des Chinesischen Restsatzes \Secret Sharing mittels CRT

  8. Analyse \ Zufallsanalyse \ 3D-Visualisierung
