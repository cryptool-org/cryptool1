RC2 (Menü Analyse \ Symmetrische Verschlüsselung (modern))

Für die Verschlüsselung mit dem RC2-Verschlüsselungsverfahren steht ein Angriff (Brute-Force Angriff) zur Verfügung.

Der zu durchsuchende Schlüsselraum muss im Dialog Suchbereich für Schlüssel eingegeben werden. Nachdem das Dokument mit allen Schlüsseln entschlüsselt worden ist, wird als Klartext das Dokument vermutet, welches in den ersten 1000 Zeichen die kleinste Entropie hat.

Im Kapitel Szenarien befindet sich ein Beispiel zum Angriff auf das Triple DES-Verschlüsselungsverfahren im CBC-Modus. Dieses Szenario kann für jedes symmetrische Verschlüsselungsverfahren durchgespielt werden, jedoch werden die verschlüsselten Dokumente immer unterschiedlich sein.

Voraussetzungen an das Dokument für einen erfolgreichen Angriff: siehe Brute-Force-Angriff.
