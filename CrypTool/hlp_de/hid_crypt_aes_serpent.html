AES-Kandidat Serpent (Menü Ver-/Entschlüsseln \ Symmetrisch)

Das Serpent-Verschlüsselungsverfahren arbeitet mit einer Blocklänge von 128 Bit und einer Schlüssellänge von 256 Bit, wobei kürzere Schlüssel auf 256 Bit aufgefüllt werden.

Die Eingabe des Schlüssels erfolgt im Dialog Schlüsseleingabe.

In CrypTool ist ein Brute-Force Angriff für das Serpent-Verschlüsselungsverfahren enthalten.
