Inverse Permutation

Eine Permutation der Länge n besteht aus den n Elementen einer Menge. Nutzt man als Menge die Zahlen 1 bis n und reiht diese Zahlen in einer beliebigen Folge ohne Wiederholung auf, hat man eine bestimmte Permutation (kann auch als Vektor betrachtet werden).

Eine inverse Permutation ist eine Permutation, in der jedes Element und die Stelle, an der sich das Element befindet, vertauscht werden. Beispielsweise sind die beiden nachfolgenden Permutationen invers zueinander:

P1 = ( 3, 6, 5, 2, 7, 1, 8, 4 )

P2 = ( 6, 4, 1, 8, 3, 2, 5, 7 )

Will man zu einer gegebenen Permutation die Inverse bestimmen, so muss jedes Element der Permutation einzeln betrachtet werden.

Um die Inverse zu P1 zu bestimmen, sucht man zunächst diejenige Stelle in P1, an der sich das kleinste Element (hier: Element "1") befindet. Beginnend mit eins zählt man die Permutation durch und erhält als Ergebnis die Stelle (Index) "6".

Oben gilt also: 1 = P1(6) und 6 = P2(1) = Index von P1(6).

Das erste Element von P1 steht an sechster Stelle, daher ist das erste Element von P2 gleich sechs. Das zweite Element von P1 steht an vierter Stelle, daher ist das zweite Element von P2 gleich vier.

Analog dazu wird verfahren, bis alle Elemente der Permutation behandelt wurden.
