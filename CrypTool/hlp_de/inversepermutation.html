Inverse Permutation

Eine inverse Permutation ist eine Permutation, in der jedes Element und die Stelle, an der sich das Element befindet, vertauscht werden. Beispielsweise sind die beiden nachfolgenden Permutationen invers zueinander:

P1 = { 3, 6, 5, 2, 7, 1, 8, 4 }

P2 = { 6, 4, 1, 8, 3, 2, 5, 7 }

Will man zu einer gegebenen Permutation die Inverse bestimmen, so muss jedes Element der Permutation einzeln betrachtet werden. Das nachfolgende Beispiel zeigt, wie aus der Permutation P1 die zugehörige Inverse P2 erzeugt wird.

Um die Inverse zu P1 zu bestimmen, sucht man zunächst diejenige Stelle in P1, an der sich das erste Element (hier: Element "1") befindet. Beginnend mit eins zählt man die Permutation durch und erhält als Ergebnis die Stelle "6". Anders ausgedrückt: Das erste Element der Permutation steht an der sechsten Stelle.

Zur Überprüfung: Das erste Element von P1 steht an sechster Stelle, daher ist das erste Element von P2 gleich sechs. Das zweite Element von P1 steht an vierter Stelle, daher ist das zweite Element von P2 gleich vier.

Analog dazu wird verfahren, bis alle Elemente der Permutation behandelt wurden.
