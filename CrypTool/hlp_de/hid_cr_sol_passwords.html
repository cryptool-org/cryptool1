Lösung zum Szenario Passwort

Eine Lösung bei diesem Szenario besteht darin, das Passwort abzuhören und dann selbst zu verwenden. Dazu gehen Sie wie folgt vor:

  1. Erstellen Sie eine Verbindung zwischen dem Angreifer und Router 1 oder Router 2.

  2. Greifen Sie Router 1 oder Router 2 an. (Er muss jetzt rot erscheinen.)

  3. Starten Sie das Protokoll vom Client aus („Authenfizieren des Clients“). Sobald das Datenpaket bei Router 1 oder Router 2 weitergeleitet wird, schickt er Ihnen eine Kopie davon. Dadurch erhalten Sie das Passwort.

  4. Jetzt können Sie das Protokoll vom Angreifer aus starten („Authenfizieren des Angreifers“) und werden authentifiziert, da Sie das Passwort kennen.
