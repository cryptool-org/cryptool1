Dialog Vorbereitungen durchführen

Dieser Dialog wird angezeigt, wenn Sie im Dialog für den Seitenkanalangriff auf das Hybridverschlüsselungsprotokoll die Schaltfläche Vorbereitungen durchführen klicken.

Im dargestellten Szenario möchte Alice eine hybridverschlüsselte Datei an Bob schicken. Sie kann entweder selbst eine Datei hybridverschlüsseln, oder sie kann eine bereits hybridverschlüsselte Datei auswählen.

Weitere Informationen zum Thema Hybridverschlüsselung finden Sie hier.

Bemerkung:

Wenn Sie selbst die Hybridverschlüsselung nicht durchführen wollen und auch keine hybridverschlüsselte Nachricht auswählen, wird eine in CrypTool integrierte Nachricht verwendet.
