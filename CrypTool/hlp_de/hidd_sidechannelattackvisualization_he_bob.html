Dialog Aktueller Status von Bob

Sie können diesen Dialog aufrufen, indem Sie im Dialog Seitenkanalangriff auf das Hybridverschlüsselungsprotokoll auf das Symbol von Bob klicken.

Durch diesen Dialog erhalten Sie detaillierte Informationen über Bob.

Neben dem Portrait von Bob befindet sich das Aktionsprotokoll, das alle wichtigen Schritte protokolliert, die Bob während des Seitenkanalangriffs durchgeführt hat.

Darunter können Sie einstellen, durch welches Schlüsselwort Bob erkennt, ob eine Nachricht tatsächlich von Alice stammt. Weitere Informationen zu diesem Mechanismus finden Sie hier.

In einer Liste darunter sieht man in entschlüsselter Form alle Sessionkeys, die Bobs Server im bisherigen Verlauf des Seitenkanalangriffs empfangen hat. Neben der Reihenfolge lässt sich aus der Tabelle das jeweilige Entschlüsselungsergebnis feststellen.
