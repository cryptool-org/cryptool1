Elliptische Kurven

CrypTool benutzt Elliptische Kurven-Verfahren zum Signieren von Dokumenten (Menü Digitale Signaturen/PKI). Nähere Informationen zu Elliptischen Kurven finden Sie im CT-Buch.

Wie Sie ein Dokument mit einem Elliptischen Kurven-Schlüssel signieren

Über das Menü Schlüsselverwaltung \ Schlüssel erzeugen gelangen Sie zu folgendem Dialog:

Füllen Sie die Pflichtfelder Name, Vorname, PIN-Code und PIN-Verifikation aus und achten Sie darauf, dass bei Verfahren der Radiobutton Elliptische Kurven aktiviert ist! Quittieren Sie die Eingabe, indem Sie auf Neues Schlüsselpaar generieren klicken.

Im Menü Digitale Signaturen können Sie über Signieren bzw. Signatur überprüfen ein Dokument signieren bzw. die "Echtheit" der Signatur überprüfen. Das Szenario RSA-Verschlüsselungsverfahren (Signatur) können Sie auch mit Elliptische-Kurven-Schlüsseln durchspielen.
