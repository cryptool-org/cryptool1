Befehle des Menüs Symmetrische Verschlüsselung (modern) (Menü Analyse)

Das Menü Symmetrische Verschlüsselung (modern) enthält folgende Befehle:

IDEA Automatische Analyse für IDEA-Verschlüsselungsverfahren.
RC2 Automatische Analyse für RC2-Verschlüsselungsverfahren.
RC4 Automatische Analyse für RC4-Verschlüsselungsverfahren.
DES (ECB) Automatische Analyse für DES-Verschlüsselungsverfahren im ECB-Modus.
DES (CBC) Automatische Analyse für DES-Verschlüsselungsverfahren im CBC-Modus.
Triple DES (ECB) Automatische Analyse für Triple DES-Verschlüsselungsverfahren im ECB-Modus.
Triple DES (CBC) Automatische Analyse für Triple DES-Verschlüsselungsverfahren im CBC-Modus.
MARS Automatische Analyse für MARS-Verschlüsselungsverfahren.
RC6 Automatische Analyse für RC6-Verschlüsselungsverfahren.
Rijndael Automatische Analyse für Rijndael-Verschlüsselungsverfahren.
Serpent Automatische Analyse für Serpent-Verschlüsselungsverfahren.
Twofish Automatische Analyse für Twofish-Verschlüsselungsverfahren.
