Die Modulo-Operation

Die mathematische Operation modulo bestimmt den Rest bei der Division einer Zahl n durch eine andere Zahl k. Das Ergebnis von n modulo k liegt also im Bereich zwischen 0 und k-1.

Beispiel:

0 modulo 3 = 0 1 modulo 3 = 1 2 modulo 3 = 2
3 modulo 3 = 0 4 modulo 3 = 1 5 modulo 3 = 2
6 modulo 3 = 0 ...
