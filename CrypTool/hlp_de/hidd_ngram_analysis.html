Dialog N-Gramm-Liste

Sie erreichen diesen Dialog über den Menüeintrag Analyse \ Werkzeuge zur Analyse \ N-Gramm.

Dieser Dialog zeigt Ihnen die Liste der häufigsten N-Gramme (die 1 bis 5000 häufigsten N-Gramme, je nach Einstellung unter Anzeige der [ ] häufigsten N-Gramme). Sie können als Analyse auswählen zwischen Histogramm, Digramm, Trigramm und N-Gramm (4 <= N <= 16).

Der Button Liste berechnen sortiert die Liste nach den Häufigkeiten und gibt sie in der Tabelle aus:

  • Die Spalte Nr. gibt an, das wievielt häufigste N-Gramm in dieser Tabellenzeile steht,
  • in der Spalte Zeichenfolge steht welches N-Gramm betrachtet wird und
  • die beiden folgenden Spalten geben die relative Häufigkeit (in %) und die absolute Häufigkeit (wie oft kommt das N-Gramm im analysierten Dokument vor) an.

Die aktuell berechnete N-Gramm-Liste können Sie mit dem Button Liste speichern in ein Textfenster in CrypTool übertragen.

Über den Button Textoptionen können Sie das Alphabet einstellen, das bei der Analyse verwendet wird. N-Gramme, die aus nicht im Alphabet enthaltenen Zeichen bestehen, werden nicht in der Tabelle angezeigt.
