Befehle des Menüs Einzelverfahren

Das Menü Einzelverfahren enthält folgende Befehle:

Hashwerte Berechnet den Hashwert oder Fingerabdruck für ein Dokument.
Schlüssel aus Passwort generieren Erzeugt aus einem Passwort einen sicheren Schlüssel.
Komprimieren Komprimiert / dekomprimiert Daten mit dem ZIP-Verfahren.
RSA-Demo Eine praktische Demonstration zu den Themen Primzahlen, RSA und Faktorisieren von Zahlen.
Zufallsdaten erzeugen Erzeugt "zufällige Daten" mit Hilfe von Pseudo-Zufallsgeneratoren.
ASN.1-Decodieren einer Datei Decodiert eine im ASN.1-Format vorliegende Datei.
Diffie-Hellman-Demo Visualisiert, wie zwei Kommunikationspartner über einen öffentlichen Kanal mit diesem Verfahren einen geheimen Schlüssel aushandeln können.
Angriff auf den Hashwert einer digitalen Signatur Demonstriert, wie man durch das Bestimmen einer Hashkollision eine digitale Signatur perfekt fälscht.

Die Untermenü-Einträge Hashwerte und Komprimieren sind nur dann sichtbar, wenn im Hauptfenster eine Datei aktiv ist, d.h. wenn wenigstens eine Datei geöffnet ist.
