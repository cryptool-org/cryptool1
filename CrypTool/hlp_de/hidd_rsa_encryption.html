Dialog RSA-Verschlüsselung

Sie können diesen Dialog über den Menüeintrag Ver-/Entschlüsseln \ Asymmetrisch \ RSA-Verschlüsselung aufrufen.

Das aktive Dokument wird mit dem RSA-Verfahren verschlüsselt. Es werden alle verfügbaren RSA-Schlüsselpaare angezeigt. Aus dieser Liste wählen Sie den Empfänger. Dessen öffentlicher Schlüssel wird benutzt, um die Nachricht zu verschlüsseln. Eine Entschlüsselung ist nur mit dem dazugehörigen geheimen Schlüssel möglich.
