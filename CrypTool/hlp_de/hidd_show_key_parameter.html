Dialog Öffentliche Parameter (für RSA)

Zu dieser Hilfe gelangt man auf 2 Weisen:

  1. Man kreuzt in dem Dialogfenster Erzeugen eines asymmetrischen Schlüsselpaares die Option Anzeigen des erzeugten Schlüsselpaares an und drückt dann Neues Schlüsselpaar erzeugen. Der öffentliche und der geheime Schlüssel werden mit Hilfe von Pseudozufallszahlen generiert.

    Daraufhin öffnet sich der Dialog "Öffentliche Parameter":

    Das aktuell angezeigte Schlüsselpaar wird durch Drücken von Übernehmen gespeichert.



  2. Im Dialogfenster Verfügbare asymmetrische Schlüsselpaare wird ein Schlüssel ausgewählt. Drückt man den Button Öffentliche Parameter anzeigen, so öffnet sich der Dialog "Öffentlicher Schlüssel von...". Durch Drücken von F1 erhält man diese Hilfe.
