Dialog Automatische Analyse (ermittelten Schlüssel ausgeben)

Dieser Dialog gehört zur automatischen Analyse der Verschlüsselungsverfahren

  • Vigenère,

  • byteweise Addition und

  • binäres exklusives Oder (XOR).

In diesem Dialog wird der ermittelte Schlüssel ausgegeben. Sofern er falsch ist, kann er geändert werden.

Die Betätigung des Entschlüsseln-Knopfes führt zur Ausgabe des ermittelten Klartextes, Abbrechen beendet die Analyse.
