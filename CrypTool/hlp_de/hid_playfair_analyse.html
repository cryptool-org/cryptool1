Playfair (Menü Analyse \ Symmetrische Verschlüsselung (klassisch) \ Manuelle Analyse)

Über diesen Menüeintrag können Sie in einem Dialog eine manuelle Analyse der Playfair-Verschlüsselung durchführen.

Für die Playfair-Verschlüsselung steht im Moment noch kein automatischer Angriff zur Verfügung. Deshalb steht das Tool Analysenhilfe zur Verfügung, mit dem Sie eine Playfair-verschlüsselte Datei manuell analysieren können.

Falls einige Zuordnungen von verschlüsselten Digrammen und Klartext-Digrammen bekannt sind oder erraten werden können, kann man mit dieser Analysenhilfe den Schlüssel ermitteln und der ganzen Text entschlüsseln.

Im Kapitel Szenarien befindet sich ein Beispiel zum Angriff auf das Playfair- Verschlüsselungsverfahren.
