Dialog Angriff auf den Hashwert der digitalen Signatur

Sie erreichen diesen Dialog über den Menüeintrag Analyse \ Hashverfahren \ Angriff auf den Hashwert der digitalen Signatur.

Der auf dem Geburtstagsparadoxon beruhende generische Angriff auf den Hashwert der digitalen Signatur kann gegen jede beliebige Hashfunktion durchgeführt werden. Mit ihm verfolgt der Angreifer das Ziel, zwei unterschiedliche Nachrichten mit gleichem Hashwert zu finden.

Er konzipiert zunächst eine harmlose Nachricht, die das Opfer des Angriffs ohne Bedenken digital signieren würde, sowie eine gefährliche Nachricht, die es auf keinen Fall elektronisch unterschreibt. Der Angreifer erzeugt nun eine große Anzahl von Modifikationen der beiden Nachrichten, die zwar inhaltlich mit den Ursprungsnachrichten identisch sind, sich jedoch z.B. durch zusätzliche Leerzeichen von ihnen unterscheiden.

Findet ein Angreifer ein Paar aus harmloser/gefährlicher Nachricht mit identischem Hashwert, so lässt er die harmlose Nachricht von seinem Opfer signieren und tauscht anschließend die harmlose gegen die gefährliche Nachricht aus.

In den Gruppierungen "Harmlose" Datei auswählen bzw. "Gefährliche" Datei auswählen legt der Angreifer die beiden Nachrichten fest, mit denen der Geburtstagsangriff durchgeführt werden soll.

Danach können noch Optionen für den Angriff auf den Hashwert der digitalen Signatur festgelegt werden.

Gestartet wird der Angriff mittels Nachrichtenpaar suchen.

Bemerkung 1:

Wenn Sie zwei identische Dateien angeben, wird ebenfalls versucht, durch Anhängen von verschiedenen "Mustern" zwei verschiedene Dateien mit gleichem Hashwert zu erzeugen.

Bemerkung 2 (in CrypTool mitgelieferte Beispieldateien):

Das CrypTool-Paket enthält die Beispieldateien "Original.txt" und "Faelschung.txt" als Beispiel einer harmlosen Nachricht und einer gefährlichen Nachricht. Falls Sie eigene Dateien benutzt haben, danach aber wieder die Pfade dieser Beispieldaten angezeigt haben wollen, löschen Sie in der CrypTool Registry-Datei die Zeilen 

SignatureAttackHarmlessFile

SignatureAttackDangerousFile.
