Dialog Vigenère (Analyse nach Schrödel)

Sie erreichen diesen Dialog über den Menüeintrag Analyse \ Symmetrische Verschlüsselung (klassisch) \ Ciphertext-Only \ Vigenère (Analyse nach Schrödel).

Sobald Sie die Analyse über den Button Analyse starten gestartet haben, werden mögliche Lösungen (Paare aus Schlüssel und zugehörigem Klartext) in die Liste geschrieben. Sobald die Analyse beendet ist, können Sie über den Button Analyseergebnisse anzeigen detaillierte Informationen über die Analyse anzeigen lassen. Beachten Sie bitte, dass die Anzeige der Analyseergebnisse nicht funktioniert, falls Sie die Analyse vorzeitig abbrechen.

Allgemeine Informationen in Bezug auf die Analyse nach Schrödel finden Sie hier.
