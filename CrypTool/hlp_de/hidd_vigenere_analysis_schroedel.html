Dialog Vigenère (Analyse nach Schrödel)

Sie erreichen diesen Dialog über den Menüeintrag Analyse \ Symmetrische Verschlüsselung (klassisch) \ Ciphertext-Only \ Vigenère (Analyse nach Schrödel).

Sobald Sie die Analyse über den Button Analyse starten gestartet haben, werden mögliche Lösungen (Paare aus Schlüssel und zugehörigem Klartext) in die Liste geschrieben. Über den Button Analyse abbrechen können Sie die Analyse vorzeitig beenden.

Klicken Sie den Button Analyseergebnisse anzeigen, werden die Ergebnisse der Analyse in einem neuen Dokument angezeigt. Der Analyse-Dialog wird dabei geschlossen.

Allgemeine Informationen in Bezug auf die Analyse nach Schrödel finden Sie hier.
