Menü Hashverfahren (Menü Einzelverfahren)

Das Menü Hashverfahren enthält folgende Befehle:

MD2 Berechnet einen Hashwert mittels MD2.
MD4 Berechnet einen Hashwert mittels MD4.
MD5 Berechnet einen Hashwert mittels MD5.
SHA Berechnet einen Hashwert mittels SHA.
SHA-1 Berechnet einen Hashwert mittels SHA-1.
SHA-256 Berechnet einen Hashwert mittels SHA-256.
SHA-512 Berechnet einen Hashwert mittels SHA-512.
RIPEMD-160 Berechnet einen Hashwert mittels RIPEMD-160.
Hashwert einer Datei Berechnet einen Hashwert einer Datei.
Hash-Demo Eine praktische Demonstration zu Hashfunktionen.
Schlüssel aus Passwort generieren (PKCS #5) Erzeugt aus einem Passwort einen sicheren Schlüssel.
Generieren von HMACs Erzeugt verschiedene Keyed-Hash Message Authentication Codes (HMACs).
