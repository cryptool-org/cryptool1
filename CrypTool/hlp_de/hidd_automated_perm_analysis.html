Dialog Analyse einer einstufigen Spaltentransposition (Known-Plaintext)

Sie erreichen diesen Dialog über den Menüeintrag Analyse \ Symmetrische Verschlüsselung (klassisch) \ Known-Plaintext \ Einstufige Spaltentransposition.

In den Dialog müssen die benötigten Daten zur automatische Analyse (Known-Plaintext-Angriff) für das Permutation-Verschlüsselungsverfahren eingegeben werden.

In diesen Dialog müssen Sie den Klartext und den verschlüsselten Text auswählen.

Bei der Analyse der Spaltentransposition wird der Text normalerweise in eine Tabelle der Breite n (= Länge der Permutation) zeilenweise eingetragen. Das Chiffrat wird nun spaltenweise in der von der Permutation vorgegebenen Reihenfolge ausgelesen. In CrypTool besteht neben der klassischen Variante die Möglichkeit, den Klartext wahlweise spalten- oder zeilenweise ein- oder auszulesen. Des Weiteren besteht die Möglichkeit zu bestimmen, ob die eigentliche Permutation spaltenweise oder zeilenweise ausgeführt wird. Standard ist hier spaltenweise, in der Literatur wird jedoch gelegentlich eine zeilenweise Notation eingesetzt. Werden bei allen 3 Optionen (Einlesen, Permutation und Auslesen) die Werte zwischen zeilenweise und spaltenweise getauscht, so führt dies zum gleichen Ergebnis.

Außerdem haben Sie die Möglichkeit, die Länge des zu suchenden Schlüssels einzuschränken. Entweder Sie geben ein Intervall oder genau eine feste Schlüssellänge an. Sofern Sie die Länge des Schlüssels kennen, können Sie das Programm anweisen, die Analyse nur für diese Schlüssellänge zu versuchen. Bei großen Dokumenten führt dies zu einer Zeitersparnis. Beachten Sie dabei aber, dass zum Beispiel ein Schlüssel der Länge 3 nicht gefunden wird, wenn Sie die Länge auf 1 bis 2 einschränken.

Durch Betätigen des Knopfes Schlüssel berechnen wird die Analyse gestartet; die Analyse wird nicht gestartet, wenn Sie den Dialog mit dem Abbrechen-Knopf beenden.

Sofern der Schlüssel ermittelt werden konnte, wird er im Dialog Schlüsselausgabe: Spaltentransposition (einstufig) ausgegeben, ansonsten erfolgt eine Meldung, dass der Schlüssel nicht gefunden werden konnte.
