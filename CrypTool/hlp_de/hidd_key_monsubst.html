Dialog Schlüsseleingabe: Monoalphabetische Substitution / Atbash

Sie erreichen den Dialog über den Menüeintrag Ver-/Entschlüsseln \ Symmetrisch (klassisch) \ Substitution/Atbash.

In diesem Dialog wird der Schlüssel für die monoalphabetische Substitution eingegeben.

Bei der Eingabe des Schlüssels sind folgende Punkte zu beachten:

  • Das Eingabefeld akzeptiert nur die 26 Buchstaben. Alle anderen Eingaben werden automatisch zurückgewiesen, das heißt nicht angezeigt.
  • Es wird nicht zwischen Groß- und Kleinschreibung unterschieden, das heißt Kleinbuchstaben werden in Großbuchstaben umgewandelt.

Es gibt zwei Möglichkeiten, den Schlüssel einzugeben:

  • Sie können die gewünschte Permutation (eine Anordnung aller 26 Alphabetzeichen) direkt eingeben. In diesem Fall ist ein 26 Zeichen langer Text einzugeben. Das A wird dann auf den ersten Buchstaben des eingegebenen Textes abgebildet, das B auf den zweiten, und so fort. Hierbei ist zu beachten, dass im eingegebenen Text keine Buchstaben doppelt vorkommen.
  • Sie können aber auch ein Schlüsselwort eingeben, aus dem die Permutation abgeleitet wird. Ein Beispiel für diesen Fall: Es ist das Schlüsselwort "EISENBAHN" eingegeben. Die resultierende Permutation ermittelt sich daraus wie folgt: Das A wird auf den ersten Buchstaben (in diesem Falle das E abgebildet). Entsprechend wird das B auf I und das C auf S abgebildet. Da der vierte Buchstabe des Schlüsselwortes, das E, schon einmal aufgetaucht ist, wird es übersprungen (man erinnere sich daran, dass keine zwei (verschiedenen) Buchstaben auf den gleichen Buchstaben abgebildet werden dürfen). Das geht so lange weiter, bis alle Buchstaben des Schlüsselwortes abgearbeitet sind, das heißt es wird D auf N, E auf B, F auf A und G auf H abgebildet. Alle restlichen Buchstaben (H bis Z) werden auf die noch nicht benutzten Buchstaben in umgekehrter Reihenfolge abgebildet, das heißt H auf Z, I auf Y, ..., Z auf C.

Falls Sie das Dokument mit dem eingegebenen Schlüssel verschlüsseln wollen, klicken Sie den Knopf Verschlüsseln. Zum entschlüsseln wählen Sie den Knopf Entschlüsseln.

Die Formatierung (Beibehalten der Leerzeichen, Sonderzeichen und Groß- und Kleinschreibung) der Ausgabe des ver- bzw. entschlüsselten Dokuments kann im Dialog Textoptionen eingestellt werden.
