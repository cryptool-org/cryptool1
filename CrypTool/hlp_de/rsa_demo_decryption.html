RSA-Entschlüsselung einer Nachricht

Nur wenn Sie den geheimen RSA-Schlüssel d besitzen, können Sie eine Nachricht entschlüsseln. Wenn Sie nur die öffentlichen RSA-Parameter N und e kennen, dann gibt es (für zu kurze RSA-Schlüssel) die Möglichkeit durch den Faktorisierungsangriff den geheimen RSA-Schlüssel zu berechnen.

Zur Entschlüsselung von RSA-verschlüsselten Nachrichten geben Sie die Zahlencodierung der verschlüsselten Nachricht ein. Nach dem Klick auf Entschlüsseln wird die Eingabe

c = c[1] # c[2] # .... # c[j]

mit dem geheimen Schlüssel (N,d) nach der Formel

m[i] = c[i]^d mod N für i = 1, ..., j

entschlüsselt. Danach erfolgt die Umwandlung der Zahlen m = m[1] # m[2] # .... # m[j] in die ursprüngliche Textnachricht M, indem die Aufteilung in Blöcke rückgängig gemacht wird:
