Dialog Sichere Primzahl erzeugen

Dieser Dialog dient zum Erzeugen einer Primzahl p der Form p = 2q+1. Wobei die Zahl q wieder eine Primzahl ist. Die Primzahl q wird Sophie Germain Primzahl genannt und p hat die Eigenschaft einer sicheren Primzahl. Nach einem Klick auf die Schaltfläche Primzahl generieren wird eine zufällig gewählte Primzahl mit der gewählten Bitlänge bestimmt.

Sichere Primzahlen erfüllen wichtige Eigenschaften für Kryptographiesysteme deren Sicherheit auf das diskrete Logarithmusproblem basiert. Z.B. das Diffie-Helman Schlüsselaustausch Protokoll. Die multiplikative Gruppe modulo p besitzt dann eine Untergruppe mit maximaler Primzahlordnung.
