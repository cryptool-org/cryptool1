Dialog Sichere Primzahl erzeugen

Dieser Dialog dient zum Erzeugen einer Primzahl p der Form p = 2q+1, wobei die Zahl q ebenfalls eine Primzahl ist.
- Die Primzahl q wird Sophie-Germain-Primzahl genannt.
- Die Primzahl p hat dann die Eigenschaft einer "sicheren Primzahl".

Nach einem Klick auf die Schaltfläche Primzahl generieren wird eine zufällig gewählte Primzahl p mit der gewählten Bitlänge bestimmt.

Sichere Primzahlen p erfüllen wichtige Eigenschaften für Kryptographiesysteme, deren Sicherheit auf dem diskreten Logarithmusproblem basiert (z.B. das Diffie-Hellman-Schlüsselaustausch-Protokoll.
Die multiplikative Gruppe modulo p besitzt dann eine Untergruppe mit maximaler Primzahlordnung.
