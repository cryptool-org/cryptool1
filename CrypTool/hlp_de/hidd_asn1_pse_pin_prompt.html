Dialog ASN.1-PIN-Eingabe

Diesen Dialog erreichen Sie über das Menü Einzelverfahren \ ASN.1-Decodieren einer Datei.

Durch diesen Dialog wird sichergestellt, dass nur autorisierte Personen Zugriff auf weitere Daten innerhalb einer ASN.1-Struktur erhalten.

Falls Sie die richtige PIN eingeben, werden die in der ASN.1-Struktur enthaltenen Daten in einem neuen Fenster angezeigt.

Geben Sie eine falsche PIN ein, so bleibt es lediglich bei der Ausgabe des ASN.1-Dumps.

Bemerkung:

Der Dialog wird ausschließlich dann angezeigt, wenn die PSE, auf die zugegriffen wird, auch wirklich PIN-geschützt ist. Ansonsten erfolgt der Zugriff auf das Zertifikat und dessen Ausgabe ohne weitere Eingaben des Benutzers.
