Dialog RSA-Entschlüsselung

Sie können diesen Dialog über den Menüeintrag Ver-/Entschlüsseln \ Asymmetrisch \ RSA-Entschlüsselung aufrufen.

Wenn das aktive Dokument die Ausgabe einer RSA-Verschlüsselung enthält, kann man daraus den Klartext zurückerhalten, wenn man den PIN-Code des Empfängers kennt. Mit diesem PIN-Code wird auf den geheimen Schlüssel, der zum Entschlüsseln der Nachricht gebraucht wird, zugegriffen (der geheime Schlüssel befindet sich verschlüsselt in der PSE).
