Dialog Optionen Analyse monoalphabetische Substitution

In diesem Dialog werden Einstellungen für die automatische Analyse der monoalphabetischen Substitution gemacht.

Sofern eine automatische Analyse gewünscht ist, muss ausgewählt werden, ob der Klartext Deutsch oder Englisch ist. Dies geschieht durch Anklicken des entsprechenden Feldes.

Zwei Optionen können ausgewählt werden:

e als häufigstes Zeichen annehmen.

Erweiterte Analyse (für deutschen Klartext).

Eine Option ist aktiviert, wenn sich ein Häkchen in dem Feld vor dem Text befindet. Das Aktivieren bzw. Deaktivieren geschieht durch einen Mausklick. Nach der Analyse öffnet sich der Dialog Manuelle Nachbearbeitung.

Ist die automatische Analyse nicht gewünscht, sondern soll der Schlüssel manuell gefunden werden, so ist die Option Manuelle Analyse anzuwählen. Dadurch wird die automatische Analyse übersprungen und der Dialog Manuelle Nachbearbeitung öffnet sich sofort.

Es werden keine Zuweisungen (Substitutionen) vorgenommen. Alle Zuweisungen sind

auf unbestimmt (*) gesetzt und müssen von Ihnen eingegeben werden.

Weitere Erklärungen zu den Einstellungen im Dialog Analyse monoalphabetische Substitution befinden sich im Kapitel automatische Analyse der monoalphabetischen Substitution.
