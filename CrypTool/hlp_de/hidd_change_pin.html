Dialog PIN-Eingabe für Import/Export einer PSE

In diesem Dialog können die nötigen PINs zum Export bzw. Import von PSEs eingegeben werden.

Für den Export einer PSE in das PKCS #12 Format geben Sie oben die PIN der PSE ein, die exportiert werden soll. Weiter unten kann die PIN gewählt werden, mit der die exportierte PKCS #12 Datei geschützt wird. Unter Verifikation muss diese PIN zum Schutz vor Tippfehlern wiederholt werden.

Beim Import von PKCS #12 Dateien in eine (neu erstellte) PSE geben sie oben die PIN der PKCS #12 Datei ein. Weiter unten kann die PIN für die PSE gewählt werden. Unter Verifikation muss diese PIN wiederum zum Schutz vor Tippfehlern wiederholt werden.

Die Taste OK startet dann den Export bzw. Import, während Abbrechen Sie zum vorherigen Dialog zurückführt.
