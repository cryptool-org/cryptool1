Hypothese H0

In der Statistik ist die Hypothese H0 eine Behauptung über die Verteilung von einer oder mehrerer Zufallsvariablen. Die alternative Hypothese H1 besagt, dass H0 nicht wahr ist.

Der Test einer Hypothese ist ein auf beobachtete Werte von Zufallsvariablen basierendes Verfahren, das zur Ablehnung von H0 (also zur Annahme von H1) oder zur Akzeptanz von H0 (und zur Ablehnung von H1) führt.
