Menü Tools (Menü Einzelverfahren)

Das Menü Tools enthält folgende Befehle und Untermenüs:

Codierungen Anwendung verschiedener Codierungen.
Komprimieren Komprimieren und Entkomprimieren von Dokumenten mittels ZIP.
Zufallsdaten erzeugen Erzeugung pseudo-zufälliger Daten mittels verschiedener Zufallsgeneratoren.
Passwort-Qualitätsmesser Misst die Qualität von Passwörtern.
Passwort mit bestimmter Entropie erzeugen Erzeugt Passwörter mit einer bestimmten Entropie.
Länger einer Zahl Berechnet die Länge einer Zahl in verschiedenen Darstellungen.

Der Untermenüeintrag Komprimieren ist nur dann sichtbar, wenn im Hauptfenster eine Datei aktiv ist, d.h. wenn wenigstens eine Datei geöffnet ist.
