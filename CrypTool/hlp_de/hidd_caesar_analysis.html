Dialog Automatische Analyse Caesar-Verschlüsselungsverfahren

Dieser Dialog gehört zur automatischen Analyse des Caesar-Verschlüsselungsverfahrens.

In diesem Dialog wird der ermittelte Schlüssel ausgegeben. Sofern er falsch ist, kann er geändert werden.

Die Betätigung des Entschlüsseln-Knopfes führt zur Ausgabe des Klartextes, Abbrechen beendet die Analyse.
