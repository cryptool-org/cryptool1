Dialog Solitaire-Analyse

Dieser Dialog dient zur Analyse (Solitaire-Analyse) eines mit dem Solitaire-Algorithmus verschlüsselten Geheimtextes. Der Geheimtext muss zuvor als Textdokument geladen werden. Auch hier wird der Text auf 65.535 Zeichen beschränkt und der Rest des Textes wird abgeschnitten. Dann wird anhand des gefundenen End-Decks ein Initial-Deck erzeugt. Mit Hilfe dieses Initial-Decks wird anschließend der Schlüsselstrom erzeugt, mit dem der Geheimtext entschlüsselt werden kann.

Bitte folgen Sie den Nummerierungen der umrandeten Felder in diesem Dialog (Gruppierung 1 bis 4).

Ermittlung des Initial-Decks

Gruppierung 1: Parameter des gefundenen Decks

Bitte wählen Sie die Art und Weise aus, mit der Sie das "gefundene" End-Deck bestimmen möchten. Dafür klicken Sie bitte einen der beiden Buttons End-Deck laden oder Nach Vorgabe an.

  • Button End-Deck laden: Sie können ein zuvor bei der Verschlüsselung gespeichertes End-Deck laden.
  • Button Nach Vorgabe: Sie können in der Gruppierung "Nach Vorgabe" ein neues End-Deck erzeugen. Außerdem kann die Kartenanzahl des gewünschten Kartendecks variiert werden.

Die Gruppierung Nach Vorgabe wird aktiviert, sobald Sie in Gruppierung 1 "Nach Vorgabe" gewählt haben. Zuerst wählen Sie bitte die Anzahl der gewünschten Karten (3-54). 54 ist der Defaultwert. Anschließend definieren Sie das von Ihnen gewünschte End-Deck, indem Sie die einzelnen Buttons (1-54) in der entsprechenden Reihenfolge anklicken. Unterläuft Ihnen ein Fehler, klicken Sie bitte den Reset-Button und beginnen die Eingabe erneut. Während Sie die einzelnen Buttons anklicken, können Sie Ihr bisher erzeugtes Deck im Feld "Gefundenes Deck" sehen. Sobald Sie die von Ihnen gewünschte Kartenanzahl erreicht haben, d.h. alle Buttons gedrückt sind, bekommen Sie eine entsprechende Meldung.


Gruppierung 2: Joker positionieren

Aufgrund der Irreversibilität von Solitaire muss an einigen Stellen (den Sonderfällen, wo es jeweils zwei Möglichkeiten gibt, den Joker einzuordnen) entschieden werden, ob Joker A bzw. Joker B hinten oder vorne einsortiert werden sollen, wenn versucht wird, das Initial-Deck zu ermitteln. Diese Gruppierung gibt Ihnen die Möglichkeit, zwischen Immer vorne, Immer hinten und Manuell zu wählen.

  • Immer vorne: Dabei werden die Joker bei einem Sonderfall immer vorne eingeordnet.
  • Immer hinten: Dabei werden die Joker bei einem Sonderfall immer hinten eingeordnet.
  • Manuell (=Starten): Bei einem Sonderfall wird der Benutzer um eine Entscheidung gebeten. Mit dem Button Abbrechen innerhalb des Entscheidungsdialogs kann zum Ausgangszustand zurückgekehrt werden.

Sowohl bei der manuellen als auch bei der automatischen (einheitlichen) Auswahl können falsche Wege gewählt werden, die zu einem falschen Klartext führen.


Gruppierung 3: Initial-Deck erzeugen

Diese Gruppierung zeigt drei unterschiedliche Kartendecks:

  • Gefundenes Deck: Stellt das gefundene Kartendeck dar, welches entweder selbst eingegeben oder geladen wurde. Es sollte ein gültiges End-Deck sein, das beim Verschlüsseln mit Solitaire entstand.
  • Zwischendeck: Stellt die verschiedenen Änderungen des Kartendecks während der Kryptoanalyse dar.
  • Initial-Deck: Bei einer erfolgreichen Kryptoanalyse eines mit Solitaire verschlüsselten Textes wird hier das gleiche Kartendeck angezeigt, wie jenes, welches zur Schlüsselstromerzeugung bei der Verschlüsselung benutzt wurde.

Gruppierung 4: Schlüsselstromerzeugung

Aus dem ermittelten Initial-Deck kann der Solitaire-Algorithmus mit dem Button Schlüsselstrom automatisch erzeugen gestartet werden. Damit kann der Schlüsselstrom erzeugt werden, mit dem man den Geheimtext wieder entschlüsseln kann.

Der Schlüsselstrom enthält so viele Zahlen wie das zuletzt geöffnete CT1-Fenster Zeichen enthält. Per Klick auf den Button Entschlüsseln kann der Geheimtext sofort entschlüsselt werden (vorausgesetzt das erzeugte Initial-Deck ist identisch mit demjenigen, welches zur Schlüsselstromerzeugung bei der Verschlüsselung diente). Es wird angenommen, dass das zuletzt geöffnete CT1-Fenster den Geheimtext enthielt. Der gefundene "Klartext" wird in ein neues Fenster im CT1-Hauptfenster geschrieben.


Mit Hilfe des Buttons Abbrechen können Sie den Dialog jederzeit verlassen.
