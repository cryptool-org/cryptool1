Dialog Automatische Additions- und XOR-Analyse (ermittelte Schlüssellänge ausgeben)

Sie erreichen diesen Dialog über den Menüeintrag Analyse \ Symmetrisch Verschlüsselung (klassisch) \ Ciphertext only \ Addition.

Dieser Dialog gehört zur automatischen Analyse des Verschlüsselungsverfahrens durch byteweise Addition oder durch binäres exklusives Oder (XOR).

In diesem Dialog wird die ermittelte Schlüssellänge ausgegeben. Sofern die Länge falsch ist, kann sie geändert werden. Außerdem wird erwartet, dass Sie das am häufigsten vorhandene Zeichen im Klartext eingeben.

Die Betätigung des Entschlüsseln-Knopfes führt zur Ausgabe des Klartextes, Abbrechen beendet die Analyse.
