Dialog Faktorisierungsliste (Details zur erfolgreich durchgeführten Faktorisierung)

Dieser Dialog wird angezeigt, wenn Sie aus dem Dialog Faktorisieren einer Zahl heraus die Schaltfläche Details klicken.

In diesem Dialog erhalten Sie alle Details der letzten Faktorisierung.

Im oberen Textfeld wird die faktorisierte Zahl angezeigt. Die Liste zeigt alle Zwischenschritte der Faktorisierung an:

  • welcher Zahl wurde faktorisiert,
  • die beiden Faktoren der Zerlegung,
  • die angewendete Faktorisierungsmethode und
  • die Zeitdauer der Faktorisierung.

Durch ein Doppelklick auf eine Zeile erhalten sie in der unteren Maske noch weitere Details zu dem einzelnen Faktorisierungsschritt:

  • die Bitlänge der Faktoren und
  • ob die bestimmten Faktoren Primzahlen oder zusammengesetzte Zahlen sind (diese können dann noch weiter faktorisiert werden).
