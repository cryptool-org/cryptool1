Dialog Caesar / Rot-13 Schlüsseleingabe

Sie erreichen diesen Dialog über den Menüeintrag Ver-/Entschlüsseln \ Symmetrisch (klassisch).

Mit diesem Dialog können Sie die Caesar-Verschlüsselung durchführen. Sobald Sie einen Schlüssel angegeben haben, werden das Ausgangsalphabet sowie das durch die Verschlüsselung abgebildete Alphabet angezeigt.
