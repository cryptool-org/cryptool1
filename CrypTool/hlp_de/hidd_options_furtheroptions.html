Dialog Weitere Optionen

Sie erreichen diesen Dialog über den Menüeintrag Optionen \ Weitere Optionen.

In diesem Dialog bestimmen Sie weitere Voreinstellungen für die Funktionen in CrypTool:

Optionen für die Diffie-Hellman-Demonstration

Mit der Auswahl Einleitungsdialog beim Start anzeigen bestimmen Sie, ob vor der Diffie-Hellman-Demonstration der Dialog Einleitung Diffie-Hellman-Demo erscheint.

Optionen für die Seitenkanalangriffs-Demonstration

Hier stellen Sie das Schlüsselwort ein, mit dessen Hilfe Bob erkennen kann, ob eine Nachricht tatsächlich von Alice stammt.

Optionen für die Generation von Primzahlmengen

Hier stellen Sie ein, durch welche Zeichenkette die Primzahlen in der Ausgabe getrennt werden, wenn Sie eine Menge von Primzahlen generiert haben.
