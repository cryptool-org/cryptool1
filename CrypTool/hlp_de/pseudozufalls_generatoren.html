Pseudozufallsgenerator

Ein Pseudozufallsgenerator (Pseudozufallszahlengenerator) ist ein Algorithmus, der ausgehend von einem geheimen Startwert Z_[0] - der "Saat" - eine zyklische Folge (Z_[n]) von Zahlen erzeugt. Für jemanden, der nicht die Saat Z_[0] kennt, erscheint die Sequenz Z_[i], Z_[i+1], ..., Z_[i+j] wie eine Folge von zufälligen Zahlen.
