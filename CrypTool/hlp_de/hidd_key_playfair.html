Dialog Schlüsseleingabe: Playfair

Sie erreichen diesen Dialog über den Menüeintrag Ver-/Entschlüsseln \ Symmetrisch(klassisch) \ Playfair.

Mit den Radiobuttons 5x5 Matrix und 6x6 Matrix können Sie auswählen, ob der Playfair-Algorithmus mit einer 5x5 oder 6x6 Matrix arbeiten soll. Siehe dazu die Erklärung zum Playfair-Verschlüsselungsverfahren.

Falls Sie das Dokument mit dem eingegebenen Schlüssel verschlüsseln wollen, klicken Sie den Knopf Verschlüsseln. Zum Entschlüsseln wählen Sie den Knopf Entschlüsseln.

Mit dem Knopf können Sie einen im Schlüsselspeicher gespeicherten Schlüssel direkt einfügen (falls der Knopf blass dargestellt ist, steht kein Schlüssel zur Verfügung).

In dem Feld Schlüssel können Sie ein gewünschtes Passwort eingeben.

Die Option Doppelte Zeichen im Schlüssel ignorieren beeinflusst die Art und Weise, wie das Passwort in die Matrix eingetragen wird. Ist sie aktiviert, werden doppelte Buchstaben im Passwort einfach ignoriert, andernfalls jeweils der nächste Buchstabe im Alphabet benutzt. Die unteren Kästen zeigen die aus dem eingegebenen Schlüssel erzeugte Matrix an.

Mit der Option Text vorformatieren können Sie bestimmen, ob der Text auf das benutzte Alphabet (abhängig von der Größe der Matrix) beschränkt wird. Dabei werden auch für den Fall der 5x5 Matrix alle J in I umgewandelt. Zusätzlich werden, wie in der Erklärung zum Playfair-Verschlüsselungsverfahren erwähnt, Doppelbuchstaben durch das Einfügen eines X getrennt (ein Doppel-X wird durch Einfügen eines Q getrennt). Der so veränderte Text wird zusätzlich zum verschlüsselten Text angezeigt.

Nachdem Sie den Schlüssel eingegeben haben, können Sie das Dokument ver-/entschlüsseln, indem Sie auf den Knopf Verschlüsseln oder Entschlüsseln klicken.
