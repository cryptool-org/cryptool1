Dialog Schlüsseleingabe: Hill

Sie erreichen diesen Dialog über den Menüeintrag Ver-/Entschlüsseln \ Symmetrisch(klassisch) \ Hill.

In diesem Dialog wird der Schlüssel für das Hill-Verschlüsselungsverfahren eingegeben.

Nachdem Sie im Feld Dimension die Größe des Schlüssels ausgewählt und den Schlüssel eingegeben haben, können Sie das Dokument verschlüsseln, indem Sie auf den Knopf Verschlüsseln klicken oder die Eingabetaste betätigen. Wenn Sie das Dokument entschlüsseln wollen, wählen Sie den Knopf Entschlüsseln.

Mit dem Knopf können Sie einen im Schlüsselspeicher gespeicherten Schlüssel direkt einfügen (falls der Knopf blass dargestellt ist, steht für dieses Verfahren kein Schlüssel im Schlüsselspeicher).

Alternativ können Sie die Schlüsselmatrix selbst entweder mit Zeichen des Alphabets oder mit Zahlenwerten füllen (Anmerkung: negative Zahlen sowie das Springen zwischen den Eingabefeldern per Tabulator werden nicht unterstützt). Die jeweils andere Matrixansicht wird sofort synchron aktualisiert. Dabei hat das erste Alphabetzeichen den Wert 0 oder 1. Um den Geheimtext wieder entschlüsseln zu können, müssen die Zahlenwerte der Matrix so gewählt sein, dass die Matrix invertierbar ist.
Sie können sich die Schlüsselmatrix auch per Buttonklick mit zufälligen Werten füllen lassen. Es werden so lange zufällige Eingaben gewählt, bis die resultierende Matrix invertierbar ist.

Es kann vorkommen, dass der eingegebene Schlüssel nicht zu verwenden ist:
Dies ist der Fall, wenn die dem Schlüssel entsprechende Matrix nicht invertierbar ist. Siehe dazu die Erklärung zum Hill-Verschlüsselungsverfahren. In diesem Fall erscheint ein Meldungsfenster "Schlüssel nicht zu verwenden". Dann muss ein anderer Schlüssel gewählt werden.

In diesem Dialog werden das Alphabet und der Wert des ersten Alphabetzeichens nur angeziegt. Zum Ändern sind die entsprechenden Dialoge über die Buttons Textoptionen und Weitere Hill-Optionen aufzurufen. Dort können Sie

  • definieren, ob das erste Alphabetzeichen den Wert 0 oder 1 hat,
  • definieren, mit welchem Zeichen ein Klartext aufgefüllt wird, dessen Länge kein Vielfaches der Matrixdimension ist.
