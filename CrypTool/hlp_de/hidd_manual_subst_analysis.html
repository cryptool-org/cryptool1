Dialog Substitutionsanalyse: Manuelle Nachbearbeitung

Dieser Dialog erscheint, wenn CrypTool Sie nach der Analyse einer Substitutions-Verschlüsselung zur manuellen Nachbearbeitung auffordert.

Im oberen Teil des Dialogs legen Sie die Zuordnung von Geheimtext- und Klartextbuchstaben fest. Nach jeder Veränderung wird das Ergebnis der aktuellen Entschlüsselung im Feld Aktueller Zwischenstand ausgegeben.

Lag beim Öffnen des Dialogs das Ergebnis einer automatischen Analyse vor, können Sie über den Button Ergebnis der automatischen Analyse wieder herstellen zu diesem Ergebnis zurückspringen.

Der Button Aktuellen Zwischenstand ausgeben zeigt die Entschlüsselung des Geheimtexts mit dem aktuell ausgewählten Schlüssel in einem neuen Dokument an. Beachten Sie, dass der Dialog dabei geschlossen wird.

Über den Button Schlüssel kopieren können Sie den aktuellen Schlüssel (die Zuordnung von Geheimtext- und Klartextbuchstaben) in den internen Schlüsselspeicher kopieren.
