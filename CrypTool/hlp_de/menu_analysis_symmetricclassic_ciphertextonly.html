Menü Ciphertext-Only (Menü Analyse \ Symmetrische Verschlüsselung (klassisch))

Das Menü Ciphertext-Only enthält folgende Befehle:

Caesar Automatische Analyse der Caesar-Verschlüsselung.
Vignère Automatische Analyse der Vignère-Verschlüsselung.
Vigenère (Analyse nach Schrödel) Automatische Analyse der Vignère-Verschlüsselung nach Schrödel.
ADFGVX Automatische Analyse der ADFGXV-Verschlüsselung.
Substitution Automatische Substitutionsanalyse.
Solitaire Automatische Solitaireanalyse.
Byteweise Addition Automatische Analyse für byteweise Addition.
XOR / Vernam Automatische Analyse für binäres exklusives Oder.
