Menü Manuelle Analyse (Menü Analyse \ Symmetrische Verschlüsselung (klassisch))

Das Menü Manuelle Analyse enthält folgende Befehle:

Substitution Manuelle Analyse einer Verschlüsselung durch Substitution.
Playfair Manuelle Analyse einer Verschlüsselung durch Playfair.
Solitaire Manuelle Analyse einer Verschlüsselung durch Solitaire.
