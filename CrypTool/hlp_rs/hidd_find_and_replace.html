Dialog Find and Replace

You can reach this dialog using the menu Edit \ Find/Replace.

To find or replace a search term in the present document, you can enter the search string in the current dialog and then click on the Find button.

If the search term is found, it is highlighted. If the search term cannot be found in the specified direction of search, a message box informs you.

To repeat a search, click on the Find button repeatedly. If you already closed this dialog, you can repeat the last search with the menu item Edit \ Find Next (or by pressing F3).

To replace a search term with a different text, use the button Replace. With the button Replace all you can replace all fitting search patterns.

Mode:

You can choose between two different modes: Initially, the text mode is set in which you can search strings as usual. If the hex mode is set, the search string is entered in hexadecimal format.

Search Options:

If you wish to differentiate between upper and lower case letters, then you should activate the Case sensitive checkbox.

It is also possible to specify the direction of search within the document. By default, the search proceeds downwards from the current cursor position. By activating the Find backwards checkbox the search proceeds upwards.

By activating the Regular expressions (text view only) checkbox you can search for regular expressions.

Remark 1:

Currently, regular expressions are only valid in the search field if the search is applied to windows with text documents.

Remark 2:

Information regarding regular expressions can be found here and here.

Using a short excerpt of HTML code, the following example shows how you can search and replace with the help of regular expressions.

<img width=123 height="456" src="...">
<img width=456 height="789" src="...">
<img width=789 height="123" src="...">

The task is to embed the numbers of the "width" attribute in apostrophes.

Search: width=\([0-9]+\)
Replace: width="\1"

After applying the regular expression, the code excerpt looks as follows:

<img width="123" height="456" src="...">
<img width="456" height="789" src="...">
<img width="789" height="123" src="...">
