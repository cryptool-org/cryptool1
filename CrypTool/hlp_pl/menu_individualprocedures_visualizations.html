Menu Visualization of Algorithms (Menu Individual Procedures)

The menu Visualization of Algorithms contains the following commands:

Caesar Visualization of Caesar using ANIMAL.
Vigenère Visualization of Vigènere using ANIMAL.
Nihilist Visualization of Nihilist using ANIMAL.
DES Visualization of DES using ANIMAL.
AES Visualization of AES using Flash.
Enigma Visualization of the Enigma machine using Flash.
