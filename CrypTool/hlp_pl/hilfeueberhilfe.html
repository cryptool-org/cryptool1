How to use CrypTool help

Online help is available for CrypTool using the normal Windows style. It is accessed by clicking on the terms contained in the help text which are highlighted (by being underlined and in a different color), which are hyperlinks.

Clicking on a topic that is underlined will produce a help window on that topic. You can return to the previous topic by clicking on the Back button in the help.

If you click on a topic which has a broken underscore, the current topic will remain in the window and only a small window containing a short explanation will be displayed.

In addition, the Content and Index buttons can also be used to search for subjects or terms.

There is also help on the help itself (Win help), which is designed in exactly the same way as the help for CrypTool. This can be accessed by pressing the F1 key when you have opened the CrypTool help and that is the active window, as at this moment.

The essential features of the online help are explained in more detail below

Using the menu, it is possible to print out individual pages of help and to set various options.

The online help has several facades. The Contents and Index buttons/tabs both open a dialog in which help may be obtained on the desired topic by clicking on the appropriate item.

The online help contains a help page for every dialog and every menu option, and also for all the icons on the toolbar.

Moreover, various tutorials / scenarios / examples (which can be accessed via the help index, via scenarios and via the script) can be called up, for example on encryption algorithms or prime numbers.

Opening the online help

Help for the CrypTool program can be requested in a number of different ways:

  • The help module starts up if you press F1 when CrypTool is started. On this starting page information is provided on starting work with CrypTool and the functions contained in CrypTool.
  • If the Contents button is selected from the help menu, the help index is opened. Double clicking on the entries will produce help on various topics. It is also possible to search for particular terms via the Index tab. A full-text search facility is provided on the Find tab.
  • Context-sensitive help is initiated by pressing F1. In this way one can obtain help on the active window or a particular menu option.
  • If you press the key combination of Shift+F1 or click on the help icon (this is the icon which shows a mouse pointer and a question-mark) in the toolbar, the mouse pointer will change its appearance (an additional question-mark appears). If you then click on an element in the CrypTool window, the help page which goes with that element will appear.
  • It is also possible to access the help outside of CrypTool, by double clicking on the CrypTool help file, which is located in the same folder as CrypTool.

Closing the online help

To close the help, select menu option File \ Close, click on the Close icon in the title bar or else press Esc.

When you close down CrypTool, any CrypTool help window will close automatically.
