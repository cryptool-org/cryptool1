The Vernam encryption algorithm (Menu Crypt/Decrypt \ Symmetric (classic))

The Vernam encryption algorithm works with an Exclusive-OR logical operation. However, since encryption by means of an Exclusive-OR is relatively easy to break with a short key, a document is used here as the key. XOR encryption is (provably) secure if a key (or a key file) which contains random bits and is at least as long as the document to be encrypted is chosen (this is known as a one-time pad).

To invoke this function, select Crypt/Decrypt \ Symmetric (classic) \ Vernam and a dialog in which the document with the key is selected opens.

An example of the Vernam encryption algorithm will be found in the Examples chapter.
