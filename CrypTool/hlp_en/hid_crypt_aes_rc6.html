RC6 (Menu Encrypt/Decrypt \ Symmetric (modern) \ Further Algorithms)

The RC6 encryption algorithm works with a variable block length e.g. 128 bits, and a variable key length e.g. 128, 192 and 256 bits.

CrypTool uses RC6 in CBC mode with zero initialization vector and 01-00 padding.

The key is entered in the Key entry dialog.

CrypTool contains a tool which performs a brute-force attack on the RC6 encryption algorithm.
