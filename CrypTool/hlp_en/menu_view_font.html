Menu Font (Menu View)

The menu Font contains the following commands:

Arial 8 Sets Arial 8 as font for the active text window.
Arial 10 Sets Arial 10 as font for the active text window.
Arial 12 Sets Arial 12 as font for the active text window.
Courier 8 Sets Courier 8 as font for the active text window.
Courier 10 Sets Courier 10 as font for the active text window.
Courier 12 Sets Courier 12 as font for the active text window.
