The Close menu option (system menu)

Use this menu option to close the active window or to exit CrypTool.

Double clicking on the CrypTool system menu box has the same effect as selecting the Close menu option: CrypTool closes down.

A document is closed by double-clicking on its system menu box. Similarly, a document can also be closed by selecting the Close menu option from the system menu.

Shortcuts

Title bar:
Keyboard: CTRL+F4 closes a document window.
  ALT+F4 closes the CrypTool window (i.e. used to exit the program).
