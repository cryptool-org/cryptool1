Dialog Key Entry: Hill

You can reach this dialog via the menu entry Crypt/Decrypt \ Symmetric(classic) \ Hill.

This dialog is used to enter the key for the Hill encryption algorithm.

After you have chosen the size of the key in the Dimension box and input the key, you can encrypt the document by clicking on the Encrypt button or pressing the RETURN key. If you wish to decrypt the document, just click the Decrypt button.

A key that is stored into the key memory could be inserted by the button (if there is no key stored, the button is deactivated and appears in grey).

It is possible that you could enter a key that was not in fact suitable because the matrix corresponding to the key is not invertible. (For further information, see explanation on the Hill encryption algorithm.) In this case, a "Key is unusable" message box appears. Another key must be selected.
