The CrypTool Registry"

The default settings for some options used in CrypTool to survive a program session (persistance) are stored into the CrypTool registry.

Starting CrypTool the first time introducing information (starting dialog and an example file) will be shown. Each time you have the option to decide if this will be shown in the future starts of CrypTool.

The file CrypTool registry is located in the standard Windows registry (you may open the windows registry with the DOS command regedit.exe) at the following path:

HKEY_CURRENT_USER/Software/CrypTool
