Display keys (menu Key management)

The menu Key management \ Display keys calls the dialogue Available asymmetric key pairs: here the parameters of all existing Elliptic curves, RSA and DSA keys, generated with CrypTool, can be viewed.

1)Anyone can view the domain parameters and the domain parameters associated with existing elliptic curve key pairs (created with CrypTool). The domain parameters are public information and describe the curve. If one knows the PIN which was used when the key was generated, it is possible to see all of the parameters (i.e. domain parameters, public and secret key). It is also necessary to know this PIN to delete existing key pairs.
Further information can be found in the script under "Elliptic curves in cryptography". It is explained that an elliptic curve E and a cyclic group on this curve E are specified through the domain parameters of E; this determines how valid key pairs and calculations appear on E.

2)For existing RSA and DSA key pairs which have been created using CrypTool it is possible to view a certificate. Certificates can be viewed via the Certificates dialogue.

Further information on certificates can be found in the script under the heading "Public Key Certification".
