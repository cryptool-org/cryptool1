The Analysis Menu

The Analysis menu contains the following menu items:

General General statistical information about the active document.
Random Tests Analyses the random properties of the active document.
Algorithms

(automatic analysis)

 List of algorithms for which an automatic attack is implemented.
Ciphertext-Only Automatic attacks if only the ciphertext is known.
Known Plaintext Automatic attacks if both clear and cipher text are known.
Manual Analysis Support for a user-driven attack.

The menu items General, Ciphertext-Only and Algorithms are available for both hex and text documents.

The menu items Known Plaintext and Manual Analysis are available for text documents only.

The menu item Random Tests is available for binary documents only. Random tests also deliver a kind of general information about the document.

Now the items of the single submenus and the structure of possible attacks are explained.

1. General information on the active document

Entropy Calculate the entropy of a document
Floating Frequency Calculate the floating frequency of a document
Histogram Calculate the frequency with which the various letters occur in a document
N-Gram Analysis of the frequencies of N-grams
Autocorrelation Calculate the autocorrelation of a document
Vitany View a Vitany diagram
Periodicity Analyse periodicity
Random Tests Analyse the data for properties of random data

The menu items Vitany and Random Tests are only available for binary files.

2. Automatic analysis techniques

a) Algorithms:

Algorithms Here all implemented algorithms for automatic analysis are listed.

The following chapters b) and c) distinguish the two different forms of attack Ciphertext Only and Known Plaintext. For further information, see Examples of different kinds of attack section

b) "Ciphertext Only" attacks:

Caesar Automatic analysis, Caesar encryption algorithm
Vigenère Automatic analysis, Vigenère encryption algorithm
Addition Automatic analysis, Byte Addition
XOR Automatic analysis, binary exclusive-OR
IDEA Automatic analysis, IDEA encryption algorithm
RC2 Automatic analysis, RC2 encryption algorithm
RC4 Automatic analysis, RC4 encryption algorithm
DES (ECB) Automatic analysis, DES encryption algorithm in ECB mode
DES (CBC) Automatic analysis, DES encryption algorithm in CBC mode
Triple DES (ECB) Automatic analysis, Triple-DES encryption algorithm in ECB mode
Triple DES (CBC) Automatic analysis, Triple-DES encryption algorithm in CBC mode
MARS Automatic analysis, MARS encryption algorithm
RC6 Automatic analysis, RC6 encryption algorithm
Rijndael Automatic analysis, Rijndael encryption algorithm
Serpent Automatic analysis, Serpent encryption algorithm
Twofish Automatic analysis, Twofish encryption algorithm

The Caesar analysis and Vigenère analysis are only available for text files, the remaining methods are implemented for binary files only.

c) "Known Plaintext" attacks:

Hill Automatic analysis, Hill encryption algorithm. For text files only.

3. Manual analysis techniques

Substitution Automatic analysis, Monoalphabetic substitution. For text files only.
Playfair Analysis tool, Playfair algorithm. For text files only.

4. Random Test Algorithms

Frequency Test Performs the Frequency Test on the document.
Poker Test Performs the Poker Test on the document.
Runs Test Performs the Runs Test on the document.
Serial Test Performs the Serial Test on the document.
FIPS-PUB-140-1 Test Battery Performs the FIPS-PUB-140-1 test battery on the document.

The random tests are only available for binary files.
