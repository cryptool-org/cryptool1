The Analysis Menu

The Analysis menu contains the following menu items:

General General statistical information about the active document.
Random Tests Analyses the random properties of the active document.
Algorithms

(automatic analysis)

 List of algorithms for which an automatic attack is implemented.
Ciphertext-Only Automatic attacks if only the ciphertext is known.
Known Plaintext Automatic attacks if both clear and cipher text are known.
Manual Analysis Support for a user-driven attack.

The menu items General, Ciphertext-Only and Algorithms are available for both hex and text documents.

The menu items Known Plaintext and Manual Analysis are available for text documents only.

The menu item Random Tests is available for binary documents only. Random tests also deliver a kind of general information about the document.

Now the items of the single submenus and the structure of possible attacks are explained.

  1. General information on the active document
    2. Entropy Calculate the entropy of a document
    Floating Frequency Calculate the floating frequency of a document
    Histogram Calculate the frequency with which the various letters occur in a document
    N-Gram Analysis of the frequencies of N-grams
    Autocorrelation Calculate the autocorrelation of a document
    Vitany View a Vitany diagram
    Periodicity Analyse periodicity
    Random Tests Analyse the data for properties of random data

    The menu items Vitany and Random Tests are only available for binary files.
