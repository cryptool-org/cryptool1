Commands of the top-level menu Analysis

The menu Analysis contains the following commands and sub menus:

Tools for Analysis Various tools for analysis.
Symmetric Encryption (classic) Analysis of symmetrically (classic) encrypted documents.
Symmetric Encryption (modern) Analysis of symmetrically (modern) encrypted documents.
Asymmetric Encryption Analysis of asymmetrically encrypted documents.
Hash Analysis of different hash algorithms.
Analyze Randomness Various tools to analyze randomness.
