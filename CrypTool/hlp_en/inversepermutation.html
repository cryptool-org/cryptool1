Inverse Permutation

A permutation of length n consists of the n elements of a set. If you take the numbers 1 to n of the set, and then build an arbitrary sequence (without dublication) you get a determined permutation (can also be written as a vector).

An inverse permutation is a permutation in which each element and the position of the element are exchanged. For example the two following permutations are inverse:

P1 = ( 3, 6, 5, 2, 7, 1, 8, 4 )

P2 = ( 6, 4, 1, 8, 3, 2, 5, 7 )

If you want to calculate the inverse of a permutation, you have to look at each single element separately. The following example shows how to calculate from P1 the inverse permutation of P1.

At first you determine the position at which the smallest element (here: Element "1") is located. In the above example the first element is located at position (index) "6".

It is: 1 = P1(6) and 6 = P2(1) = index of P1(6).

The first element of P1 is located at the sixth position, thus the first element of P2 is six. The second element of P1 is located at the fourth position, thus the second element of P2 is four.

Each of the permutation's elements is operated as described until all elements are done.
