Dialog Drawing Parameters

You can reach this dialog via the menu entry Options \ Plot Options

This dialog is used to specify the thickness of the lines in bar charts and graphs. These diagrams are required to output the histogram for a document. Enter the required value (between 1 and 10) into the corresponding field.

The show zero check box specifies whether the displayed diagrams should always contain the zero line or whether the display should begin with the smallest value calculated.

The field autocorrelation in relative check box is used to select the axis labels required for the autocorrelation display.

To confirm your inputs, click on Apply; to discard the changes, click on Cancel.
