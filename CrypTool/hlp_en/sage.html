SAGE scripts and notebooks about cryptography

Sage is a free open-source computer algebra system (CAS). It combines the power of many existing open-source packages into a common Python-based interface (http://www.sagemath.org).

  • Teaching with SAGE:
    David Kohel, "Cryptography", 2008,
    http://sage.math.washington.edu/home/wdj/teaching/kohel-crypto.pdf
  • Interactive introduction to elliptic curves and elliptic curve cryptography:
    Maike Massierer and the CrypTool team, "An Interactive Introduction to Elliptic Curve Cryptography", version 1.01, October 2008,
    http://sage.mathematik.uni-siegen.de:8000/home/pub/45/
