Commands of the top-level menu Individual Procedures

The menu Individual Procedures contains the following commands and sub menus:

Hash Calculate the hash value or the fingerprint for a document.
RSA Cryptosystem Various tools and demonstrations around the RSA cryptosystem.
Protocols Demonstrations of various protocols.
Chinese Remainder Theorem Applications A practical demonstration of the Chinese Remainder Theorem.
Visualization of Algorithms Visualization of various algorithms.
Secret Sharing Demonstration (Shamir) Demonstration of the Secret Sharing procedure according to Shamir.
Tools Various applications like the random generation, compression of data and password tests.
Educational Games Hands-on learning of cryptography.
Number Theory - Interactive Various interactive programs covering number theory and cryptography.
