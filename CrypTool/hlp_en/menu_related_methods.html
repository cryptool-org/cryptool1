The Individual Procedures Menu

The Individual Procedures menu contains the following options:

Hash Calculates the hash value or the fingerprint for a document.
Key Generation Generates a secure key from a password.
Compress Compresses or decompresses data using the ZIP method.
RSA Demonstration A practical demonstration on prime numbers, RSA and the factorisation of numbers.
Generate "Random Numbers" Generates "random numbers" using pseudo-random number generators.
ASN.1 Decoder Decodes an ASN.1 encoded file
Diffie-Hellman Demonstration This visual demonstration dialogue shows two communication partners agreeing on a common and secret key over an insecure line.
Hash Signature Attack Attack on the hash value of a digital signature. It demonstrates how a hash collision can be used to create a perfect fake of a digital signature.
Side-Channel Attack Demonstration Demonstrates a side-channel attack against the hybrid encryption protocol.
Challenge Response Demonstration Demonstrates several authentication-methods by means of different scenarios. By the use of these scenarios the user also gets to know the various vulnerabilities of the methods.

Message Authentication Code Generates the message authentication code of a file by means of a keyed hash function. Applications of the Chinese Remainder Theorem (CRT) A practical demonstration of the Chinese Remainder Theorem (CRT) related to the 3 topics astronomy, modular transformation and secret sharing.

The submenu items Hash and Compress are only visible if a window within the CrypTool main window is active (i.e. at least one document is shown or one file is opened).
