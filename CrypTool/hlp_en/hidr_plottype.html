Window Type Plot (in the CrypTool main window)

Within the CrypTool main window four different window types can be displayed:

  • Text (document)
  • Hex (document)
  • Plot (graphics)
  • OpenGL (graphics)

The window type Plot is used for example by:

  • Histograms
  • Floating frequency
  • Autocorrelation
  • Vitányi analysis

Remark: Active / Inactive menus
Most of the end menu items are shown in gray, if a window of type Plot is selected within the CrypTool main window -- as most of the menu items can only be applied to documents (e.g. encrypt and decrypt).
