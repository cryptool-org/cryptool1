The Edit Menu

The Edit menu contains the following options:

Undo Reverses the most recent editing action.
Cut Deletes data from the document and transfers it to the clipboard.
Copy Copies data from the document to the clipboard.
Paste Inserts data into the document from the clipboard.
Delete Deletes data from the document.
Find Searches through the document for a search term.
Find Next Continues a search for the next occurrence of the search term.
Replace Searches through the document for a search term and replaces the search term with a different text string.
Select All Selects the entire document.
Show Key Displays the key with which the document was encrypted.
Parent Window Selects the entire document.
