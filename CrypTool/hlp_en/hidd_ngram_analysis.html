Dialogue N-Gram List

This dialogue shows the list of the most frequent N-grams (the 1 to 5000 most frequent N-grams, depending on the input of Display of the [ ] most common N-Grams). You can select between Histogram, Digram, Trigram and N-Gram (4 <= N <= 16).

The button Determine List computes and sorts the N-gram list and displays it in a table:

  • The column No shows the N-gram order within the sorted list; e.g. N-gram No 5 is the 5th with respect to frequency,
  • the column Character sequence shows the N-gram and
  • the following columns shows the relative frequency (in %) and the absolute frequency (how often the N-Gram occurs in the analysed document).

The actual N-Gram list can be transferred into the CrypTool main window via the button Save list.
