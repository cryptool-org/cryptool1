Dialog N-gram List

You can access this dialog via the menu entry Analysis \ Tools for Analysis \ N-Gram.

This dialog shows the list of the most frequent n-grams in a document.

In the upper part of the Selection group you can choose between histograms, digrams, trigrams annd n-grams (4<=N<=16). The number in brackets indicates how many n-grams of the respective type are contained in the document.

In the lower part of the Selection group you can configure the upper limit for the amount of n-grams shown in the list (up to a static limit of 5000).

The button Compute list computes and sorts the n-gram list by frequency and displays it in a table:

  • The column No. shows the n-gram order within the sorted list; e.g. n-gram no 5 is the 5th with respect to frequency,
  • the column Character sequence shows the n-gram and
  • the following columns show the relative frequency (in %) and the absolute frequency (how often the n-gram occurs in the analyzed document).

The actual n-gram list can be transferred into the CrypTool main window via the button Save list.

By clicking the Text options button you can specify which alphabet is used during the analysis. N-grams that consist of non-alphabet characters are not shown in the table.
