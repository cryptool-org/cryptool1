Dialog Substitution Analysis: Manual Post-Processing

This dialog appears when CrypTool prompts you for manual post-processing after the analysis of a sustitution cipher.

In the upper part of the dialog you can define the assignments of ciphertext and cleartext characters. After each change the result of the current decryption is displayed in the field Current intermediate status of decryption.

If there was a result of an automatic analysis when the dialog was opened, you can jump back to that result using the button Reset entries to the result of the automatic analysis.

The button Show current status displays the decryption of the ciphertext with the currently chosen key in a new document. Please note that the dialog is closed in the process.

With the button Copy key you can copy the currently chosen key (the assignment of ciphertext and cleartext characters) into the internal key store.
