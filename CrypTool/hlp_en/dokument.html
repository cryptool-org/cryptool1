Document

A document is a text which is presented in whole or in part in a window. If the document will not all fit into the window, the parts that are not visible can be displayed by scrolling down through it.

Text documents are displayed by default in a window for textual input and output.

Documents containing non printable characters (hex documents) are displayed by default in a window for hexadecimal inputs and outputs.

Information on the creation and editing of a document will be found in the help index.

A document can be saved in a file.

Remark 1: Switch between text and hex display
The output after encryption or decryption of modern algorithms uses the hexadecimal window by default.
You can display a decrypted text document in the text editor window using the menu View \ Show as Text.

Remark 2: Further window types
Documents and files are displayed within CrypTool using the window types "Text" and "Hex".
Graphics prepared by CrypTool (window type Plot or OpenGL) are not considered as "documents".
