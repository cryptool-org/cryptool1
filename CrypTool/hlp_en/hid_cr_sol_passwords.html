Solution to Scenario Password

A possible solution to this scenario is to tap the password and use it yourself. You proceed as follows:

  1. You create a connection between the attacker’s computer and Router 1 or Router 2.

  2. Attack Router 1 or Router 2 (it must appear red now).

  3. You start the protocol from the client (Authenticate client). As soon as the data packet is transferred from Router1 or Router 2, a copy is forwarded to you so you can obtain the password.

  4. Now you can start the protocol from the attacker’s computer (Authenticate attacker) and you will be authenticated because you know the password.
