Verify Signature (Menu Digital Signatures/PKI)

In the Signature Verification dialog it is possible to verify a signature created using CrypTool. For this purpose the active document must contain the output which was generated using the Sign message menu option or using the button Store signature via the Signature demonstration.

To verify a signature, it is necessary to inform CrypTool manually (through appropriate selections in the Signature Verification dialog) who created a signature (or rather, with which key pair the signature was generated):

  • For RSA and DSA signatures CrypTool automatically filters out the rest of the information needed to verify the signature (such as, for example, which hash function was used to sign the message) from the signature output.
  • Under both of the elliptic curve signature algorithms (ECSP-DSA and ECSP-NR) as well as entering the name of the person who created the signature it is also necessary to select the hash function used and the procedure used. All this information will be found in the output window which opened following the selection of the Sign message menu option. Automatic recognition of the hash function used and the procedure used was deliberately omitted in order to enable experimentation.

It is possible to extract the signature by selecting menu Digital Signatures/PKI \ Extract signature.
