Alpha-Quantil

An alpha-quantile is a number Q, which separates the sorted parameter values that:

  • 100*q % of the parameter values are smaller or equal Q
  • 100*(1-q) % of the parameter values are greater Q.

Example:

The 25% quantile is the number where 25% of the parameter values are smaller or equal - and 75% of the parameter values are bigger than the stated number Q.
