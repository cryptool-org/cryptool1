Dialogue Available keys for elliptic curves

This dialogue is used to administer the parameters and asymmetric keys for elliptic curves that are stored in CrypTool. If the EC keys option field is checked, a list of elliptic curve key pairs stored is displayed. Pick the key pair you are interested in and the following options will be enabled:

  • With Show public parameters it is possible to see the domain parameters (see Elliptic Curves, Elliptic Curves and Cryptography section) and the public key held for this key pair. The values shown are display-only and cannot be altered.

  • The Show all parameters option requires entry of the PIN that was originally entered when the key pair was generated. As well as the domain parameters and the public key, the private key is also displayed. Once again, these entries cannot be altered.

  • The Delete option allows you to delete keys that are no longer used. To do this you will need the PIN that was used when the key pair was originally created.
