Dialog Authorization

This dialog can be accessed from the Available asymmetric key pairs dialog.

In that window you will have already selected an elliptic curve key pair and then pressed either the Show all parameters or the Delete pushbutton.

Both actions require proof that one has the right to execute the chosen action. This is supplied by entering the PIN which was used when the key pair was originally created. Only if the PIN is correct will the desired action be performed - otherwise an error message states that the PIN entered is incorrect.
