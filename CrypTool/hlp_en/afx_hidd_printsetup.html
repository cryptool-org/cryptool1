Dialog Print Setup

The following options are available for specifying the printer and printer port:

Printer

Select here the printer that you wish to use. Either choose the ”standard printer” or else use the ”special printer” option, and then select one of the installed printers from the list displayed. To install a new printer and/or configure the printer ports, use the Windows Control Panel.

Orientation

Choose either portrait or landscape orientation.

Paper / Size

Choose the size of the paper on which the document is to be printed.

Paper / Source

Some printers have several trays for different types of paper feed. Enter the required tray here.

Properties

Displays a dialog box in which you are offered additional selection possibilities that are specific to the printer that you wish to use.

Network...

Use this button to assign a connection to a network drive to a drive letter.
