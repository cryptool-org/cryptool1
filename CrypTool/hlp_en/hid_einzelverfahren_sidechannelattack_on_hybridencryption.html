Side-Channel Attack on "Textbook RSA" (Menu Analysis \ Asymmetric Encryption)

This is a special case of side-channel attacks. With this menu option you can start a visualization of a side-channel attack. It is based on Ulrich Kuehn's side-channel attack [Kuehn2003] against an RSA-based hybrid encryption protocol.

This attack doesn't provide the receiver's secret key but the secret session key. For this reason all messages that were sent during the respective session can be decrypted.
