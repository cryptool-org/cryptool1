Extract Signature (Menu Digital Signatures/PKI)

If the active window contains the hexadecimal output (see ASCII Table) of the document which was generated through the Sign message or using the button Store signature via the Signature demonstration menu option, then it is possible to extract the signature from the message by selecting menu Digital Signatures/PKI / Extract signature.

It should be noted that for ECSP-DSA and ECSP-NR the signature of a message consists of two numbers, c and d. If one extracts the signature from the output of one of the two procedures, the two character strings [c=] and [d=] do not belong to the actual signature – they are simply used to distinguish the two numbers c and d from each other.

When this menu option is selected, the dialog Extracted signature is displayed.
