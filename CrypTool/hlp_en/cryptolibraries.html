Used Crypto libraries

CrypTool contains an extensive collection of cryptographic algorithms and protocols.

Most of the modern cryptographic basic algorithms used are taken from:

  • the Secude library (version SDK 7.4.60) (www.secude.com) [e.g. for the hash functions, the modern symmetric and the asymmetric encryption algorithms and the key management functions],

  • the Miracl library (version 4.4.3) (http://indigo.ie/~mscott) [e.g. for the RSA demonstration, factorization],

  • the OpenSSL library (version 0.9.8b) (www.openssl.org) [e.g. for hash collisions for the signature attack],

  • the number theory library NTL of Victor Shoup (version 5.4) (http://shoup.net/ntl/) [e.g. lattice based reduction] and

  • the GNU Multiple Precision Arithmetic Library GMP (version 4.2.1) (http://www.swox.com/gmp) [e.g. for AKS].

The currently used version of the included libraries can be seen within the About CrypTool dialog (see Menu Help).
