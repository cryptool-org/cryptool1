Used Crypto libraries

CrypTool contains an extensive collection of cryptographic algorithms and protocols.

Most of the modern cryptographic basic algorithms used are taken from:

  • the Secude library (version SDK 7.4.60) (www.secude.com) [e.g. for modern symmetric and asymmetric encryption algorithms and key management functions],

  • the Miracl library (version 4.4.3) (http://indigo.ie/~mscott) [e.g. for the RSA demonstration, factorization],

  • the OpenSSL library (version 0.9.8k) (www.openssl.org) [e.g. for hash functions and hash collisions regarding the signature attack],

  • the number theory library NTL of Victor Shoup (version 5.4) (http://shoup.net/ntl/) [e.g. for lattice based reduction],

  • the GNU Multiple Precision Arithmetic Library GMP (version 4.1) (http://gmplib.org) [e.g. for AKS] and

  • the cv act library (version 1.3.0) by cv cryptovision Gmbh (http://www.cryptovision.com) for ECC methods.

The currently used version of the included libraries can be seen within the About CrypTool dialog (see Menu Help).
