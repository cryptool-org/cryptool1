Commands of the top-level menu File

The menu File contains the following commands:

New Creates a new document.
Open Opens an existing document.
Close Closes an open document.
Save Saves an open document under its file name.
Save as Saves an open document under a specified file name.
Document Properties Shows properties of the active document.
Print Prints a document.
Print Setup Used to select a printer and printer port.
Recent Files / 1,2,3... List for opening the most recently saved documents.
Exit Terminates CrypTool.
