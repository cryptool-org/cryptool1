Dialog Asymmetric Key Generation (Elliptic Curves)

This dialog is accessed by selecting the Display generated key pair option in the Generation of asymmetric key pair dialog after having generated new keys for elliptic curves.

The domain parameters selected in the Generation of asymmetric key pair dialog cannot be altered here any more. However, the user can continue creating new public and secret keys by clicking on the Generate new key pair pushbutton until he is happy with the result. It should be noted here that only the currently displayed key pair can be saved, by clicking on Apply, i.e. it is not possible to recall a key pair that has previously been displayed but was deleted by clicking on Generate new key pair.
