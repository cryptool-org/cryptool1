Tool for manual analysis of Playfair encryption algorithm (Menu Analysis \ Manual Analysis)

At present there is no automatic attack available for Playfair encryption. However, there is a tool for manual analysis.

If any assignments of encrypted and plaintext digrams are known or can be guessed, it is possible to work out the key using the analysis tool and then to decrypt the entire text.

In the Examples chapter there is an example of an attack on the encryption algorithm.
