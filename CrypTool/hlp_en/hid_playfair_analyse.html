Playfair (Menu Analysis \ Symmetric Encryption (classic) \ Manual Analysis)

Choosing this menu entry you can execute in a dialog a manual analysis of the Playfair cipher.

At present there is no automatic attack available for Playfair encryption. However, there is a tool for manual analysis.

If any assignments of encrypted digrams and plaintext digrams are known or can be guessed, it is possible to work out the key using this analysis tool and then to decrypt the entire text.

In the Examples chapter there is an example of an attack on the Playfair encryption algorithm.
