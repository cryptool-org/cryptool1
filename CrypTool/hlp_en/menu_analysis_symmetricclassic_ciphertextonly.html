Menu Ciphertext-Only (Menu Analysis \ Symmetric Encryption (classic))

The menu ciphertext-only contains the following commands:

Caesar Automatic analysis of a document encrypted with the Caesar algorithm.
Vignère Automatic analysis of a document encrypted with the Vigenère algorithm.
Vigenère (Analysis according to Schroedel) Automatic analysis of a document encrypted with the Vigenère algorithm according to Schroedel.
ADFGVX Automatic analysis of a document encrypted with the ADFGVX algorithm.
Substitution Automatic analysis of a document encrypted with the substitution algorithm.
Solitaire Automatic analysis of a document encrypted with the Solitaire algorithm.
Byte Addition Automatic analysis of a document encrypted with the byte-by-byte addition algorithm.
XOR / Vernam Automatic analysis of a document encrypted with the XOR algorithm.
