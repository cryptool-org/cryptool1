Dialog Hash function

You can reach this dialog via the menu entry Indiv. Procedures \ Hash and by klicking one of the six options.

This dialog shows the document's hash value calculated with one of the following hash functions:

MD2

MD4

MD5

SHA

SHA-1

RIPEMD-160

You can create a new window with the document‘s hash value in hexadecimal representation by clicking the Store hash value to HEX format button.
