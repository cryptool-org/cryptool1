Dialog Analysis of Monoalphabetic Substitution

This dialog is used to enter the settings for the automatic analysis of monoalphabetic substitution.

When an automatic analysis is required, the user must specify whether the plaintext is in German or English. This is done by clicking on the corresponding field.

Two options can be selected:

Use ‘e‚ as most frequent character

Extended Analysis (German plaintext)

An option is enabled if there is a check mark in the field in front of the text. The option is enabled or disabled by clicking on the field with the mouse. After the analysis has finished, the Manual Post-processing dialog appears.

If automatic analysis is not required but instead the key is to be found manually, then the Manual Analysis option should be selected in the "Analysis of Monoalphabetic Substitution" dialog. When this option is selected, the automatic analysis is omitted and the Manual Post-processing dialog opens immediately.

No assignments (substitutions) are effected. All assignments are set to not known (*) and must be entered by you.

Further information on the settings in the "Analysis of Monoalphabetic Substitution" dialog will be found in the section entitled Automatic analysis, monoalphabetic substitution.
