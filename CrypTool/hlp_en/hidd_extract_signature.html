Dialog Extracted Signature

This dialog is accessed by selecting Digital Signatures/PKI \ Extract signature.

The following extracted signature information is displayed:

  • Signer:
    Usually the name of the signer.

  • Used key:
    Here you get information about the key type and key length of the used key. Furthermore the date of key generation is shown.

  • Signature algorithm:
    Beside the signature algorithm you find the information which hash algorithm was used.

  • Signature:
    The signature is a number computed by the signature algorithm. It can be displayed in octal, decimal or hexadecimal number base or it can be displayed in binary format (hex dump plus ascii code).

  • Signed message:
    And finally you get the signed message.

Using the button Verify, you can open the dialog Verify signature to verify the extracted signature.
