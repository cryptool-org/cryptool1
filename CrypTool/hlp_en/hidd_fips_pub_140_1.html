Dialog for FIPS-PUB-140-1 test battery

You can reach this dialog via the menu entry Analysis \ Analyse Randomness \ FIPS PUB-140-1 Test Battery. The prerequisition is, that you have loaded a document of at least 2500 bytes length.

A default dialog for the FIPS-PUB-140-1 test battery looks like this:

In this dialog there are no parameters to choose from. The individual FIPS-PUB-140-1 tests are automatically performed.

Entropy points to the entropy of the file being examined. For every individual test the user is informed whether the test has been passed or not.
