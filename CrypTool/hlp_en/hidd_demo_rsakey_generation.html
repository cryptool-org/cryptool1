Dialogue Generate RSA Key

This dialogue supports the step by step generation of RSA keys. It pops up if you press the button Generate key within the RSA signature demonstration.

You only need to use this dialogue if you did not yet import a previously generated RSA key using the button Import certificate and key in the Create Certificate and PSE dialogue (button Provide certificate within the signature demonstration).

The details of how to create a RSA key is described in the online help page RSA key generation, which describes a similar functionality in the RSA cryptosystem dialogue.

In addition to what is described there, this dialogue displays the RSA modulus N‚s bit length.

Please also note the status display to the right of the entry fields for prime numbers p and q. It informs you whether the numbers entered are valid primes.

Please choose primes such that the resulting modulus is at least 300 bits long. Otherwise the document‚s hash value plus padding could be longer than the RSA modulus, which is not supported.

All RSA Parameters are automatically (re-)calculated as soon as one of the primes or the public key e is changed.

When finished press Store key if you are satisfied with the generated RSA key or Cancel to abort.
