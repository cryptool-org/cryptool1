Title bar

The title bar runs across the top edge of a window. It contains the name CrypTool and the name of the active document.

The window can be move by dragging the title bar. Please note, dialogue boxes can also be move by dragging their title bars.

A title bar can contain the following elements:

Button for the application system menu
Button for the document system menu
Minimize button
Restore button
Maximize button
Button for terminating application or closing window
Application name
Document name
