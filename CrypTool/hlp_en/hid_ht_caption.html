Title bar

The title bar runs across the top edge of a window. It contains the name CrypTool and the name of the active document.

The window can be moved by dragging the title bar. Dialog boxes can also be moved by dragging their title bars.

A title bar can contain the following elements:

  1. Button for the application system menu (left side)
  2. Button for the document system menu (left side)
  3. Minimize button
  4. Restore button
  5. Maximize button
  6. Button for terminating the application or closing the window
