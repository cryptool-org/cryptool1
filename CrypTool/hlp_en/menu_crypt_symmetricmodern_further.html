Menu Further Algorithms (Menu Crypt/Decrypt \ Symmetric (modern))

The menu Further Algorithms contains the following commands:

MARS Encryption and decryption using MARS (AES candidate).
RC6 Encryption and decryption using RC6 (AES candidate).
Serpent Encryption and decryption using Serpent (AES candidate).
Twofish Encryption and decryption using Twofish (AES candidate).
DESL Encryption and decryption using DESL.
DESX Encryption and decryption using DESX.
DESXL Encryption and decryption using DESXL.
