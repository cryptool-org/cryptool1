Significance level "alpha"

The significance level alpha of a test for hypothesis H0 gives the probability of incorrectly rejecting hypothesis H0 although it is correct. If the significance level for H0 is set too high, then the test may reject sequences that were actually generated by a random bit generator. If the significance level for a test of H0 is too low, then there is a danger that the test will also accept sequences that were not actually produced by a random bit generator. For these reasons a lot of care should be given to the choice of significance level when defining a test configuration.

Reference: Christian Schiestl, Pseudozufallszahlen in der Kryptographie, in Klagenfurt, 1999.
