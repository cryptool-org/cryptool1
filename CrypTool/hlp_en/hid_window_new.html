The New Window menu option (Menu Window)

This menu option is used to open a window which has the same content as the active window. You can open several document windows. This allows you to view different parts or views of a document simultaneously. If you change the content in one of the open windows, the changes are adopted by all the windows which contain the same document. If you open a new window, this becomes the active window and is superimposed over all the other open windows.
