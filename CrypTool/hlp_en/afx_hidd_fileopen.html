The Open dialogue box

The following fields are used to specify the file that is to be opened:

File name

Enter the file name or else select a name from the list. Only files which have the file extension selected by you in the ”Files of type” field are displayed in this field.

Files of type

Choose the type of file that is to be opened.

Look in

Choose the drive in which CrypTool has saved the file that you wish to open.

Directories

Choose the directory in which CrypTool has saved the file that you wish to open.

Network…

Use this button to assign a connection to a network drive to a drive letter.
