Dialogue ASN.1 PIN request

You can access this dialogue via the menu option Individual Procedures \ ASN.1 decoding of a file.

Authorized persons only may access the certificate contained in the PSE in case it is PIN protected.

If you enter the correct PIN, the respective certificate will be displayed in a new window. If you enter a wrong PIN, nothing but the ASN.1 dump will be displayed.

Remark:

This dialogue is only displayed if the PSE to be accessed is actually PIN protected. If it is not, no further input by the user is necessary and so the certificate is displayed right away.
