Dialog PIN Request

You can access this dialog via the menu option Individual Procedures \ Tools \ Codes \ ASN.1 decoding of a file.

Authorized persons only may access the data contained in the ASN.1 structure in case it is PIN protected.

If you enter the correct PIN, the respective certificate will be displayed in a new window. If you enter a wrong PIN, nothing but the ASN.1 hex dump will be displayed.

Remark:

This dialog is only displayed if the data (e.g. a PSE) to be accessed is actually PIN protected. If it is not, no further input by the user is necessary and so e.g. the certificate is displayed right away.
