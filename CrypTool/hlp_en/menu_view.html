The View Menu

The View menu contains the following options:

Toolbar Displays or conceals the toolbar.
Status Bar Displays or conceals the status bar.
As HexDump Shows the current window as a HexDump.
Show As Text Shows the current window as text.

The submenu items Toolbar and Status Bar are always shown.

As HexDump is displayed if a text file is active.

Show As Text is displayed only for hex files containing only printable characters.
