The Switch To menu option (application system menu)

Use this menu option to view a list of all the open applications. With this ”task list” you can move to a different application on the list or close an application.

Shortcut

Keyboard: CTRL+ESC

Dialog box options

When the Switch To menu option is selected, a dialog box is displayed in which the following options are available:

Task list

Choose the applications to which you want to switch or which you would like to close.

Switch To

Activates the selected application.

End Task

Closes the selected application.

Cancel

Closes the Task List dialog box.

Cascade

Arranges the open applications so that they overlap each other and each individual title bar is visible. This option has no effect on applications which have been reduced to the size of the icons.

Tile

Arranges the open applications in windows so that they do not overlap each other. This option has no effect on applications which have been reduced to the size of the icons.

Arrange Icons

Arranges all applications that have been minimised as icons in the lower part of the screen.
