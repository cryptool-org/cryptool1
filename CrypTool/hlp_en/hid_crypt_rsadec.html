RSA decryption (Menu Crypt \ Asymmetric)

The data in the active document should be the encrypted version of a message which has been encrypted using the RSA cryptosystem. With RSA decryption the data is restored to its original form using the RSA cryptosystem, which is accessed from the Decryption with RSA dialogue.

The Examples chapter provides an example which illustrates encryption and decryption using the RSA procedure.
