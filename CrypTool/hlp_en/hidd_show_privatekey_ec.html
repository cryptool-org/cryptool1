Dialog Public parameters and secret key

This dialog is accessed from the Available asymmetric key pairs dialog, which in turn is accessed through the menu selection Digital Signatures/PKI \ PKI \ Key Display/Export. In that window you will have already selected an elliptic curve key pair and then pressed the Show all parameters pushbutton. If the correct PIN is then entered at the prompt, this dialog opens.

The domain parameters (see Elliptic Curves, Elliptic Curves and Cryptography section) are displayed together with the public and secret keys which were stored under the above-mentioned key identifier.

The numbers (the display parameters) can be viewed in any of the octal, decimal and hexadecimal number systems (see ASCII table) (Base for presentation of numbers)
