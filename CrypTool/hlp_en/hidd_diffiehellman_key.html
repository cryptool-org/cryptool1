Dialog Key Details - Secret and Common Key of Alice and Bob

This dialog window appears after clicking the key icon from within the Diffie-Hellman Demonstration.

The dialog offers additional information on the session key, which Alice and Bob have negotiated.

Besides the key length for the keys of Alice and Bob you can see whether both separately computed keys are identical.

If the verification is successful, it is theoretically possible for Alice and Bob to use this session key to encrypt messages they send to each other.
