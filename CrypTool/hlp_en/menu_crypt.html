The Crypt Menu

The Crypt menu contains the following options:

Classical encryption algorithms

Caesar Encryption and decryption using the Caesar encryption algorithm
Vigenère Encryption and decryption using the Vigenère encryption algorithm
Hill Encryption and decryption using the Hill encryption algorithm
Substitution Encryption and decryption using monoalphabetic substitution
Homophonic Substitution Encryption and decryption using homophonic substitution
Playfair Encryption and decryption using the Playfair encryption algorithm
Addition Encryption and decryption using Byte Addition
XOR Encryption and decryption using a binary Exclusive-OR
Vernam Encryption and decryption using the Vernam encryption algorithm
Permutation Encryption and decryption using the Permutation algorithm

Symmetric encryption algorithms

IDEA Encryption and decryption using the IDEA encryption algorithm
RC2 Encryption and decryption using the RC2 encryption algorithm
RC4 Encryption and decryption using the RC4 encryption algorithm
DES (ECB) Encryption and decryption using the DES encryption algorithm in ECB mode
DES (CBC) Encryption and decryption using the DES encryption algorithm in CBC mode
Triple DES (ECB) Encryption and decryption using the Triple-DES encryption algorithm in ECB mode
Triple DES (CBC) Encryption and decryption using the Triple-DES encryption algorithm in CBC mode
MARS Encryption and decryption using the MARS encryption algorithm.(AES candidate)
RC6 Encryption and decryption using the RC6 encryption algorithm (AES candidate)
Rijndael Encryption and decryption using the Rijndael encryption algorithm (Official AES)
Serpent Encryption and decryption using the Serpent encryption algorithm (AES candidate)
Twofish Encryption and decryption using the Twofish encryption algorithm (AES candidate)
AES (self extracting) Encryption and decryption of a file into a self extracting program using the Rijndael algorithm.

Asymmetric encryption algorithms

RSA encryption Encryption using the RSA encryption algorithm
RSA decryption Decryption using the RSA encryption algorithm

Hybrid encryption demonstration

Hybrid encrypt Hybrid encryption scheme based on the encryption algorithms AES and RSA .
Hybrid decrypt Hybrid decryption based on AES and RSA.

More information on hybrid encryption can be found in the script.
