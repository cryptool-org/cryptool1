The Crypt Menu

The Crypt menu contains the following options:

Classical encryption algorithms

Caesar Encryption and decryption using Caesar.
Vigenère Encryption and decryption using Vigenère.
Hill Encryption and decryption using Hill.
Substitution Encryption and decryption using monoalphabetic substitution.
Homophonic Substitution Encryption and decryption using homophonic substitution.
Playfair Encryption and decryption using Playfair.
Addition Encryption and decryption using Byte Addition.
XOR Encryption and decryption using a binary Exclusive-OR.
Vernam Encryption and decryption using Vernam.
Permutation Encryption and decryption using permutation.

Symmetric encryption algorithms

IDEA Encryption and decryption using IDEA.
RC2 Encryption and decryption using RC2.
RC4 Encryption and decryption using RC4.
DES (ECB) Encryption and decryption using DES in ECB mode.
DES (CBC) Encryption and decryption using DES in CBC mode.
Triple DES (ECB) Encryption and decryption using Triple-DES in ECB mode.
Triple DES (CBC) Encryption and decryption using Triple-DES in CBC mode.
MARS Encryption and decryption using MARS (AES candidate).
RC6 Encryption and decryption using RC6 (AES candidate).
Rijndael Encryption and decryption using Rijndael (Official AES).
Serpent Encryption and decryption using Serpent (AES candidate).
Twofish Encryption and decryption using Twofish (AES candidate).
AES (self extracting) Encryption and decryption of a file into a self extracting program using Rijndael.

Asymmetric encryption algorithms

RSA encryption Encryption using RSA.
RSA decryption Decryption using RSA.

Hybrid encryption demonstration

Hybrid encrypt Hybrid encryption scheme based on AES and RSA.
Hybrid decrypt Hybrid decryption based on AES and RSA.

More information on hybrid encryption can be found in the script.
