Basic settings for graphic displays (Menu Options)

With the menu item Plot Options the dialogue Drawing Parameters opens: the settings for the thickness of the bars and lines in diagrams can be adjusted here.

By selecting the show zero option you can determine whether the zero line should always be visible in graphics windows or whether the y-axis should be minimal.

The autocorrelation in relative option specifies whether the figures should be presented in the autocorrelation display as relative values (as a percentage) or absolute values (in numbers).

The basic settings affect all windows which are opened after a change in the parameter settings, but not windows which were already open at the time of making this change.
