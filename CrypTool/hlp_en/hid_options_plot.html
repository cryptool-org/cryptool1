Plot Options (Menu Options)

In the dialog Drawing Parameters you can specify the width of bars and curves in bar and curve charts. These diagrams are used for displaying histograms of documents. Enter the desired value (from 1 to 10) n into the according field.

With the check box Show zero you can decide whether the histogram display shows zero at all times or it begins with the smallest value that was determined.

The check box Autocorrelation in relative is used for switching between the display of realitve values (as a percentage) and absolute values (in numbers).

The basic settings affect all windows which are opened after a change in the parameter settings, but not windows which were already open at the time of making this change.

Pushing the Apply button applies your changes to all future diagrams. Pushing the Cancel button leaves the previous settings untouched.
