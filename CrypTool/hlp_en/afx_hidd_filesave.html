Dialog Save as

The following options enable you to specify the name and location of the file that you wish to save:

File name

To save a document under a different name, enter a new file name here. A file name can consist of up to 8 characters plus a file extension of up to three characters. CrypTool adds the extension that you specified in the ”Save as type” field to the name.

Save in

Choose the drive on which the file is to be saved.

Directories

Choose the directory in which you wish to save the document.

Network…

Use this button to assign a connection to a network drive to a drive letter.
