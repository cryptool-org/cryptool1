Dialog Select hash function

The hash value - also called fingerprint - of a file depends on the selected hash method. On this dialog you are selecting the hash method (selection via the radio button list).

MD2 128 bit
MD4 128 bit
MD5 160 bit
SHA 128 bit
SHA-1 160 bit
SHA-256 256 bit
SHA-512 512 bit
RIPEMD-160 160 bit

Remind that the official PKCS #1 specification for RSA signatures (RFC 3447) does not consider the following hash functions: MD4, SHA, RIPEMD-160.
