Dialog Select hash function

The hash value - also called fingerprint - of a file depends on the selected

hash method. On this dialog you are selecting the hash method (selection via the radio button list).

MD2 128 Bit
MD4 128 Bit
MD5 160 Bit
SHA 128 Bit
SHA-1 160 Bit
RIPEMD-160 160 Bit

Remind that the PKCS #1 specification for RSA signatures does not consider the MD4 hash method .
