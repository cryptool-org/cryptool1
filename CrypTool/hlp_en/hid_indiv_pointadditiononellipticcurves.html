Point Addition on Elliptic Curves (Menu Individual Procedures \ Number Theory - Interactive)

This program allows you to generate various elliptic curves and to carry out point additions on these curves.

As number spaces you can use in this demonstration either the real numbers or groups over the prime numbers ranging from 3 to 97.

By opening the non-modal window for the log output you can simultaneously track your actions (like consecutive point additions).





Adding a point P multiple times to itself results in another point R = n*P on the same curve. For cryptography the multiple of a point on a discrete field is interesting. If working on a discrete field over a prime number, and given P and R, the problem is to find the according scalar factor n if numbers are big enough. In the screenshot above, R = 10*P. If you have only R and P, it isn't hard to find this n, as in the screenshot the calculations where done modulo p=23. Real-world cryptography is performed with numbers much bigger, where p has a length of e.g. 200 bit and more. Then it is a hard problem to find such an n.
