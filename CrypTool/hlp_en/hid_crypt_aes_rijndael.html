The Rijndael encryption algorithm (AES winner) (Menu Crypt/Decrypt \ Symmetric (classic))

The Rijndael encryption algorithm works with a variable block length e.g. 128 bits, and a variable key length e.g. 128, 192 and 256 bits.

The key is entered in the Key entry dialogue.

CrypTool contains a Brute-force attack on the Rijndael encryption algorithm.

The Rijndael Algorithm has been proposed for FIPS standardisation. Therefore it will be the successor of the antiquated Data Encryption Standard (DES) algorithm (see AES winner).
