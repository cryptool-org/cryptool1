AES (CBC) (Menu Encrypt/Decrypt \ Symmetric (modern))

The AES (CBC) encryption algorithm (also known as Rijndael) works with a variable block length e.g. 128 bits, and a variable key length e.g. 128, 192 and 256 bits.

CrypTool uses Rijndael in CBC mode with zero initialization vector and 01-00 padding.

The key is entered in the Key entry dialog.

CrypTool contains a brute-force attack on the Rijndael encryption algorithm.

The Rijndael Algorithm has been proposed for FIPS standardisation. Therefore it will be the successor of the antiquated Data Encryption Standard (DES) algorithm (see AES winner).
