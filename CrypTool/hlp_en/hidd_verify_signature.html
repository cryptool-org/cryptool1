Dialog Signature verification

This dialog enables a signature to be checked. The active document must contain the output which was generated by selecting the Sign message menu option or by selecting the Digital Signatures/PKI \ Signature Demonstration (Signature Generation) menu option (otherwise an error message is displayed).

From the list of signature creators the user selects the one with which the signature was created.

If an RSA or DSA key pair is selected, the user only has to click on Verify signature for the signature verification process to be initiated. The results of verification are displayed in a dialog.

If an elliptic curve key pair was selected, the correct Verification algorithm and Verification hash function must also be selected, otherwise the signature will be deemed to be incorrect. For further information on this point see the explanation provided on the menu option Digital Signatures/PKI \ Verify Signature.

It is also possible to view the intermediate steps involved in verification of the signature by checking the Display intermediate results option. These are displayed in the Signature verification - step by step dialog.
