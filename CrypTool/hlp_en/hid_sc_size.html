The Size menu option (system menu)

When this menu option is selected, a cross symbol containing four arrowheads is displayed with which you can change the size of the active window by pressing the CURSOR KEYS.

After the pointer has been transformed into a cross containing four arrowheads,

  • Press on one of the CURSOR KEYS (one of the keys LEFT ARROW, RIGHT ARROW, UP ARROW or DOWN ARROW) to move the pointer to the edge which you wish to move.

  • Press on a CURSOR KEY to move the frame.

  • Press the ENTER key when the window is the required size.

Please note, however, this menu option is not available when the window is already its maximum size.

Shortcut

Mouse: Drag the resize bars in the corners or on the edges of the window.
