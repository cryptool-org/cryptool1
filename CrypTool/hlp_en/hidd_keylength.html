Dialog Automatic analysis of the Vigenère encryption algorithm (show calculated key length)

You can reach this dialog via the menu entry Analysis \ Symmetric Encryption(classic) \ Ciphertext-Only \ Vigenère.

This dialog is part of the automatic analysis of the Vigenère encryption algorithm.

The calculated key length is displayed in this dialog. If the length is incorrect, it can be altered.

To resume the analysis, click on Continue; to terminate the analysis click on Cancel.
