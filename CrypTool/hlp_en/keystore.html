The Internal Key Store

In CrypTool a clipboard-like key store is implemented for each encryption method.

a) Storage of a key

If you have encrypted a document, please click the icon "display key". In the following dialog box please click the icon "copy key". Thus the desired key is stored together with the encryption method which was used to encrypt the document.

b) Retrieval of a key

If you want to decrypt or analyze an encrypted document, please click the icon "paste key". Thus the desired key is retrieved from the internal key store. Note that you can only retrieve keys from the key store if you have stored at least one key before.
