The Internal Key Store

In CrypTool a clipboard-like internal key store is implemented for each encryption method.

a) Storage of a key
If you encrypted a document, please click the icon "display key". In the following dialog box please click the icon "copy key". Thus the desired key which was used to encrypt the document is stored for the specific encryption method in the internal key store.

b) Retrieval of a key
If you want to decrypt or analyze an encrypted document, please click the icon "paste key". Thus the desired key is retrieved from the internal key store and copied into the field for key entry. Note that you can only retrieve keys from the key store if you stored a key before for the specific encryption method.

Remark: If you close CrypTool, the contents of the key store is lost.
