Dialog Message Authentication Code

This dialog can be called via the menu Individual Procedures \ Message Authentication Code.

Within this dialog you can create the message authentication code (MAC) of an opened textfile. For this purpose, you can choose between several hash functions. The generated MAC is shown in hexadecimal representation.

You have the choice between the hash functions MD2, MD4, MD5, SHA, SHA-1 and RIPEMD-160.

The secret key that is necessary for the computation can be entered manually. In addition, you can choose between different forms of combining file and secret key [Menezes, Alfred J.; van Oorschot, Paul C.; Vanstone, Scott A.: Hand-book of Applied Cryptography]. The variable m represents the message, k represents the key and H stands for the hash function.

The text window displays the combination that underlies the calculation.

Remark:

Please note that this dialog is only available for text files and that the file length is limited to 16,000 characters. Characters going beyond this limitation are ignored.
