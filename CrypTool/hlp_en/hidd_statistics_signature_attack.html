Dialogue Statistics of the attack

If the attack was successful some statistical data about the attack will be displayed in the Dialogue Statistics of the attack.

The category Efforts made to find a pair of messages displays:

  • The Calculation time broken down into years, days, hours, minutes and seconds.
  • The number of steps that were required to find a pair of messages with the same hash value.
  • The total number of hash operations.
  • A list which breaks down each run on Steps until collision, Check of the collision and Total steps.
The category Additional bytes shows how many bytes were attached to the harmless and dangerous message so that the hash values of the resulting modifications collided. If the method Attach characters was chosen the number of attached bytes is the same for both messages, whereas using the method Insert blanks can result in a different number of attached bytes for each message.

By Clicking Print statistics all collected data is printed in a text file and another attack can be started in the dialogue Attack on the hash value of the digital signature.

The same applies to clicking Cancel except that no text file is printed.
