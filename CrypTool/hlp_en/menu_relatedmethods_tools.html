Menu Tools (Menu Individual Procedures)

The menu Tools contains the following commands and sub menus:

Codes Applications of different codings.
Compress Compresses or decompresses data using the ZIP method.
Generate Random Numbers Generates "random numbers" using pseudo-random number generators.
Password Quality Meter Measures the quality of passwords.
Password Entropy Generates passwords with a specific entropy.

The sub menu Compress is only visible if a window within the CrypTool main window is active (i.e. at least one document is shown or one file is opened).
