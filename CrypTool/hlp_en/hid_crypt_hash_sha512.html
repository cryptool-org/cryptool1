SHA-512 hash value (Menu Individual Procedures \ Hash)

Like SHA-224, SHA-256 and SHA-384, the SHA-512 algorithm (Secure Hash Algorithm) is part of the so called SHA-2 family of hash functions. SHA-512 yields a hash value of 512 bit (64 byte) and is the successor of the SHA-1 algorithm.

As of today, the NIST recommends the SHA-2 family: "Federal agencies should stop using SHA-1 for digital signatures, digital time stamping and other applications that require collision resistance as soon as practical, and must use the SHA-2 family of hash functions for these applications after 2010." Critical applications already employ SHA-2 algorithms (i.e. TrueCrypt 5.x uses RIPEMD-160, SHA-512 and Whirlpool).

This algorithm family is described as Secure Hash Standard (SHS) in FIPS 180-2 and FIPS 180-3.

Similar to the AES competition, a successor to SHA-2 is sought after with the help of NIST workshops.
