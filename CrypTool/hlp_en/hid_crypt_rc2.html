RC2 (Menu Encrypt/Decrypt \ Symmetric (modern))

RC2 is a symmetric encryption algorithm with variable key length. It was developed by Ron Rivest (RC stands for Ron‚s Code or Rivest Cipher). RC2 is a 64-bit block cipher. Its speed does not depend on the key length. The algorithm itself has never been published officially.

In CrypTool, RC2 is used in CBC mode with zero initialization vector and PKCS#7 padding.

The key is entered in the Key entry dialog.

CrypTool contains a tool which performs a brute-force attack on the RC2 encryption algorithm.
