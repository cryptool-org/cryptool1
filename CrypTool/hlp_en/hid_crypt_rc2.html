The RC2 encryption algorithm (Menu Crypt/Decrypt \ Symmetric (modern))

RC2 is a symmetric encryption algorithm with variable key length. It was developed by Ron Rivest (RC stands for Ron‚s Code or Rivest Cipher). RC2 is a 64-bit block cipher. Its speed does not depend on the key length. The algorithm itself has never been published. In CrypTool, RC2 is integrated in the CBC mode.

The key is entered in the Key entry dialogue.

CrypTool contains a tool which performs a Brute-force attack on the RC2 encryption algorithm.
