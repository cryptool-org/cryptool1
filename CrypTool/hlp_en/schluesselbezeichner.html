Key identifier

Every key pair generated and any additional relevant data are stored under a key identifier. This is made up as follows (<abc> means that abc is optional, and may therefore be omitted):

RSA key pair:

CN=Last name First name <[Key ID]> [RSA KEY], L=Frankfurt, O=DB, C=DE

DSA key pair:

CN=Last name First name <[Key ID]> [DSA KEY], L=Frankfurt, O=DB, C=DE

Elliptic curve key pair:

Last name First name <[Key ID]> [curveID primeXXXvY]
