Dialogue Automatic Analysis (Byte Addition and Exclusive-OR)

This dialogue is part of the automatic analysis of the encryption algorithm using Byte Addition or Binary Exclusive-OR.

The calculated key length is displayed in this dialogue. If the length is incorrect, it can be altered. The user is also required to enter the character which is expected to occur the most often in the plaintext.

To view the plaintext, click on Decrypt; to terminate the analysis click on Cancel.
