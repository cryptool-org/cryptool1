Menu Symmetric Encryption (classic) (Menu Analysis)

The menu Symmetric Encryption (classic) contains the following commands and sub menus:

Ciphertext-Only Automatic analysis, when only the encrypted text is given.
Known Plaintext Automatic analysis, when cleartext and ciphertext are given.
Manual Analysis User controlled analysis.



In CrypTool 2 (CT2) you can find some more implementations of attacks against classic ciphers.

Some very good new cryptanalytic results against classic ciphers can be found in the journal Cryptologia, like:

  • George Lasry, Nils Kopal, and ArnoWacker. Solving the Double Transposition Challenge with a Divide-and-Conquer Approach. Cryptologia, 38(3):197–214, 2014.
  • George Lasry, Ingo Niebel, Nils Kopal, and Arno Wacker. Deciphering ADFGVX Messages from the Eastern Front of World War I. Cryptologia, 41(2):101–136, 2017.
