Print Preview (Menu File)

Use this menu option to display the active document as it will look when printed out. When this menu option is selected, the main window is replaced by a Print Preview window in which one or two sides of the document are presented in print format. In the Print Preview toolbar options are provided enabling you to specify whether you wish to print one or two pages at a time. Moreover, you can also move forwards and backwards in the document, view pages magnified or reduced in size or initiate a print job.
