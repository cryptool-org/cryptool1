Close (Menu File)

Use this menu option to close the active document. CrypTool will prompt you to save the changes to your document before closing it. If you close a document without saving it, all the changes you have made since the document was last saved will be discarded. Before a document that does not have a name is closed, CrypTool displays the Save As dialog box and prompts you to give the document a name and save it.
