Dialogue Public parameters

This dialogue is accessed from the Available asymmetric key pairs dialogue, which in turn is accessed through the menu selection Key management \ Display keys. In that window you will have already selected an elliptic curve key pair and then pressed the Show public parameters pushbutton.

The domain parameters (see Elliptic Curves, Elliptic Curves and Cryptography section) are displayed together with the public key which was stored for this key pair.

The numbers (the displayed parameters) can be viewed in any of the octal, decimal and hexadecimal number systems (see ASCII table) (Base for presentation of numbers)
