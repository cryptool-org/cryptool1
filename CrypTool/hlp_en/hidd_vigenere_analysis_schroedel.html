Dialog Vigenère (Analysis according to Schroedel)

You can reach this dialog via the menu entry Analysis \ Symmetric Encryption (classic) \ Ciphertext-Only \ Vigenère (Analysis according to Schroedel).

After clicking the button Start analysis, possible results (pairs of a key and the respective cleartext) are written into the list. When the analysis is done, you can display detailed information on the analysis by clicking the button Show analysis results. Please note that the display of detailed information does not work if you cancel the analysis.

General information regarding the analysis according to Schroedel can be obtained here.
