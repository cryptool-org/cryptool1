Dialog Compute Mersenne Numbers

You can reach this dialog via the menu entry Individual Procedures \ Number Theory - Interactive \ Compute Mersenne Numbers.

This dialog can be used to calculate powers where both base and exponent are integers. This dialog calculates the exact number very fast, even when powers become very big. The shown result is the power minus 1 (as usual for calculating Mersenne numbers).

Status July 2010: For example, the currently biggest known prime is a Mersenne number (base = 2; exponent = 43,112,609), which was found in the GIMPS project (Great Internet Mersenne Prime Search).
If base and exponent are known CrypTool calculates this result being more than 12 milllion decimal digits long in less than 20 seconds on a normal PC.
