Dialog Automatic analysis of the Vigenère encryption algorithm (show key)

You can reach this dialog via the menu entry Analysis \ Symmetric Encryption(classic) \ Ciphertext-Only \ Vigenère.

This dialog is part of the automatic analysis of the Vigenère encryption algorithm.

The key which CrypTool has worked out is displayed in this dialog. If it is incorrect, it can be altered.

To view the ascertained plaintext, click on Decrypt; to terminate the analysis click on Cancel.
