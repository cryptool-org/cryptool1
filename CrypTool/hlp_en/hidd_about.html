Dialog About CrypTool

You can reach this dialog via the menu entry Help \ About CrypTool.

This dialog can also be accessed by clicking on the icon in the toolbar.

This dialog states the copyright, the sponsors and the version number.

Clicking the button Contributors will show the authors of CrypTool.

Clicking the button Translations will show information about the translations of CrypTool into other languages than English and German.

Remark:

The stated version of the Microsoft compiler used to build CrypTool is just an internal version (i.e. "MSC 15.00" instead of "Visual Studio 2008").
