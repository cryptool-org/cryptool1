ECC-AES Encryption (Menu Encrypt/Decrypt \ Hybrid)

The hybrid encryption system applies to the active document a symmetric cipher (here AES) using a session key.

Then the session key is encrypted with an asymmetric cipher (here with an ECC key selected in the dialog Hybrid Encryption with ECC-AES).

The created ciphertext can be decrypted using the dialog hybrid decryption with ECC-AES.
