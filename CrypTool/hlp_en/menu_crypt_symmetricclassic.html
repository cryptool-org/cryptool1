Menu Symmetric (classic) (Menu Encrypt/Decrypt)

All classic encryption methods belong to the group of symmetric encryption methods, where sender and receiver are using the same key.

The menu Symmetric (classic) contains the following commands, in order to encrypt or decrypt data:

Caesar / Rot-13 Encryption and decryption using Caesar.
Vigenère Encryption and decryption using Vigenère.
Hill Encryption and decryption using Hill.
Substitution / Atbash Encryption and decryption using monoalphabetic substitution.
Playfair Encryption and decryption using Playfair.
ADFGVX Encryption and decryption using ADFGVX.
Byte Addition Encryption and decryption using Byte Addition.
XOR Encryption and decryption using a binary Exclusive-OR (XOR).
Vernam / OTP Encryption and decryption using Vernam.
Homophone Encryption and decryption using homophonic substitution.
Permutation / Transposition Encryption and decryption using permutation.
Solitaire Encryption and decryption using Solitaire.
Scytale / Rail Fence Encryption and decryption using simple transpositions.
