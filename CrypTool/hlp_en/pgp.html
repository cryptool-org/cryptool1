What is PGP?

PGP (short for Pretty Good Privacy) is the standard for privacy and secure communication in the Internet in the private area. The software package was originally developed by Phil Zimmermann and provides encryption and digital signature for emails and files. The first version of PGP was available as early as June 5th 1991. It is well-known for helping to defend the right and freedom of private electronic communication.

Status in June 2002:

The company NAI had bought PGP and developed the Windows version. After releasing the client version 7.03 NAI stopped developing and supporting the program any further. NAI neither could find a new owner nor were they willing to give their version to the Internet community as open source till then. Phil Zimmermann said, now (his) PGP is accessible for no one anymore – captured in a "prison of intellectual property".
Because no PGP source code had been released since version 2.6.3, an open source project was started: GPG = GnuPG (GNU privacy guard). Here an open source program is developed which also implements the OpenPGP standard (RFC 2440), which was developed based on PGP, and is therefore compatible to PGP. GnuPG version 1.0 was published in 1999 and founds the basis of other open source PGP packages.

URL: www.gnupg.org


The German government [German Federal Ministry of Economics and Technology (BMWA/BMWi ) and the German Federal Ministry of the Interior (BMI)] started to fund the command line-oriented project GnuPG (www.gnupp.de) intensively – with the goal to offer a packet named GnuPP increasing the user friendliness, especially on the Windows platform, by adding a graphical user interface and plug-ins.

Remark:
The GnuPP initiative for a free graphical frontend is no longer active. The last update of the project web site dates September 2003.


Status in February 2003:

The new company PGP corporation, of which Phil Zimmermann is also a stake holder, bought PGP back. English version of PGP 8.0 was released for Windows (incl. Windows XP) and Macintosh (Instructions in German can be found at www.helmbold.de/pgp). There is a version for non-commercial usage and the sources are also published again. Phil Zimmermann emphasized in a talk given at January 30, 2003: As long as he can decide, the source code will continue to be published and there will be no intended trapdoors within PGP.

URL: www.pgp.com


The current version of the open source package GnuPP is 1.1 – also available for all Windows versions (including Windows XP). It is based on GnuPG 1.06 (the base system GnuPG already achieved version 1.2.1 and is also available for Linux).
The package GnuPP can encrypt directly files, e-mails, the clipboard or the content of the active window.
GnuPP can be used as freeware both privately and commercially. The software offers a German and an English interface with 2 introductions: one for beginners and one for experienced users. The assistant GPA is especially helpful for users without crypto-experience in order to generate and manage keys.
As PGP 8.0 does GnuPP deliver many plug-ins for email programs (like Outlook, The Bat, Eudora and Pegasus).


Status in December 2006:

Since 2005 release 9.0 of PGP Desktop is available. This release features instant messaging (AOL) communication encryption and supports two factor authentication via smart cards. New product lines extend PGP's range of applications (e.g. PGP Whole Disk Encryption or PGP Support Package for BlackBerry) and build up the PGP Encryption Platform. In an official letter to the PGP Community CEO, Philip Dunkelberger, strengthens PGP's commitment to continue to publish the source code in order to enable the community to conduct code reviews and to maintain customer trust. However, the source code is mainly available for the product PGP Desktop, and even there the build tools are not included in the package. These build tools can only be obtained at charge via PGP Professional Services. With regard to the available sources PGP speaks of more than 3.000 downloads per month. Besides the founder Phil Zimmermann, PGP's technical advisory board consists of some known experts such as Bruce Schneier or Dr. Crispin Cowan.

The GnuPG project (www.gnupg.org) develops the free basis software (the actual signing and encryption software) as a command line program for several operating systems. The GnuPG project is very active and continuously publishes new versions: Since beginning of December 2006 GnuPG 1.4.6 is available.
