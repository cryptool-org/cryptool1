Dialog Display Hill Key Matrix

This dialog displays the key used to encrypt the plaintext that has been found by automatic analysis of the Hill encryption algorithm.

If the Encrypt field is activated (i.e. the radial button next to that option is enabled) the key used for encryption is displayed. A field can be activated by a mouse click. On the other hand if the Decrypt option is activated, the key used for decryption is displayed. This is useful in order to avoid having to repeat the Known Plaintext attack if by mistake you entered the plaintext as the encrypted text (or vice versa) in the Hill cipher Analysis dialog. On clicking this field, the key entered is displayed.

This dialog is also used to display the key with which a document is encrypted when the Hill encryption algorithm has been used (see Show Key function). The keys for all the other encryption algorithms are displayed in the Key dialog.
