The Print dialogue box

The following options enable you to specify how the document should be printed:

Select Printer

The active printer and printer port are displayed here. Choose the Setup tab to change the printer and the printer port.

Setup

Displays a Print Setup dialogue box in which you can choose a printer and a printer port.

Page Range

Specify the pages that you wish to print out:

All Prints the entire document
Selection Prints the highlighted text
Pages Prints the range of pages that you specify in the fields ”From” and ”To”.

Number of copies

Specify how many copies of the defined page range should be printed.

Collate

Prints the copies in the sequence of the page numbers rather than separately printing several copies for each page.

Print Quality

Specifies the print quality. Generally, the lower the print quality the faster it is to print a document.
