Vigenère (Analysis according to Schroedel) (Menu Analysis \ Symmetric Encryption (classic) \ Ciphertext-Only)

There are methods for breaking Vigenère ciphers if the key length is known. However, these methods require the ciphertext to be of a certain minimum length and contain repetitions.

Schrodel's method overcomes these restrictions with a different approach using logic, digram frequency and language properties.

Unfortunately, Schroedel's method will fail if the key is not a dictionary word, and the analysis may be very time-consuming under certain conditions. Therefore, the quality of the underlying statistical data (digram and trigram statistics as well as a dictionary) is essential for the success of the method. Using the dialog Analysis Options, you can specify which files are to be used for the analysis.

In the Analysis Options dialog you can also adjust the Threshold of the analysis. The smaller the value, the more possible solutions (keys) are shown during analysis. Depending on the quality of the statistical data (see above), it might make sense to vary this threshold.

Literature:
