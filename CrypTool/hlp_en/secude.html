Secude IT Security Ltd

Secude IT Security Ltd, Darmstadt, provides modern cryptographic solutions to end users and developers.

Secude IT Security Ltd (http://secude.com/) was originally involved with the implementation of the elliptic curves in CrypTool and supplied the modern cryptosystems from its library (Secude library).

The Secude library is certified for the SAP interfaces SNC (Secure Network Communications) and SSF (Secure Store and Forward).
