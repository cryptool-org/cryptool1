Visualizations in CrypTool (Overview)

Visualizations (interactive demonstrations) are implemented for the following topics:

  1. creating an electronic signature,

  2. RSA demonstration,

  3. processing hybrid encryption with RSA or ECC and AES,

  4. how changes to a document change the hash value

    1. sensitivity of hash algorithms,

    2. generation of hash value collisions (birthday paradox),

  5. processing the Diffie-Hellman key exchange,

  6. authentication protocols,

  7. side-channel attack,

  8. shared secret using

    1. Chinese Remainder Theorem,
    2. Shamir,

  9. visualization of algorithms (Caesar, Vigenère, Nihilist, DES, AES, Enigma),

  10. 3D visualization of big sets of random numbers,

  11. password quality meter,

  12. password entropy.

  13. learning tool for number theory,

  14. point addition on Elliptic Curves.

You can access to these visualizations via the following menu items:

  1. Individual Procedures \ RSA Cryptosystem \ Signature Demonstration (Signature Generation)

  2. Individual Procedures \ RSA Cryptosystem \ RSA Demonstration

  3. Crypt \ Hybrid

  4. How changes of a document change the hash value

    1. Individual Procedures \ Hash \ Hash Demonstration

    2. Individual Procedures \ Attack on the hash value of the digital signature

  5. Individual Procedures \ Diffie-Hellman Demonstration

  6. Analysis \ Asymmetric Encryption \ Side-Channel Attack on "Textbook RSA"

  7. Individual Procedures \ Protocols \ Network Authentication

  8. Shared Secret

    1. Individual Procedures \ Secret Sharing Demonstration (Shamir)

    2. Individual Procedures \ Chinese Remainder Theorem Applications \Secret Sharing by CRT

  9. Individual Procedures \ Visualization of Algorithms

  10. Analysis \ Analyze Randomness \ 3D Visualization

  11. Individual Procedures \ Tools \ Password Quality Meter

  12. Individual Procedures \ Number Theory - Interactive \ Learning Tool for Number Theory

  13. Individual Procedures \ Number Theory - Interactive \ Point Addition on Elliptic Curves
