Dialog Brute Force Analysis

The brute-force analysis decrypted the given ciphertext with all possible keys out of the selected key space and then calculated for each valid decrypted message the entropy value. A decrypted message is valid only if all decrypted characters are part of the character set preset (you have the opportunity to define the character set here: Menu / Options / Analysis Options.)

This list contains the decrypted messages with the lowest entropy values.

Especially in case of very short ciphertexts it could be, that there are wrong candidates with a small entropy value. Here you can select the candidate with the correct decryption.
