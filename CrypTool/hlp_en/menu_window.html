Commands of the top-level menu Window

The menu Window offers the following options which enable you to view several documents in the application window, arranged in different ways:

Cascade Arranges the windows so that they overlap.
Tile Arranges the windows favorably next to each other.
Arrange Icons Arranges the windows that have been minimised to icons.
Close All Closes all open windows.
Windows 1,2,3... Activates the relevant window.
Windows All windows are shown in a list, if there are more then 9 windows open.
