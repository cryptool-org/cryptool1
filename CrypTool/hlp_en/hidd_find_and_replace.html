Dialog Find and Replace

To find or replace a search term in the present document, you can enter the search string in the current dialog and then click on the Find button.

If the search term is found, it is highlighted. If the search term cannot be found in the specified direction of search, a message box informs you.

To repeat a search, click on the Find button repeatedly. If you already closed this dialog, you can repeat the last search with the menu item Edit \ Find Next (F3).

To replace a search term with a different text, use the button Replace. With the button Replace All you can replace all fitting search patterns.

Search Options:

If you wish to differentiate between upper and lower case letters, then you should activate the Case sensitive checkbox.

It is also possible to specify the direction of search within the document. By default, the search proceeds downwards from the current cursor position. By activating the Find backwars checkbox the search proceeds upwards.

By activating the Regular expressions (text view only) checkbox you can search for regular expressions.

Bemerkung:

Currently, regular expressions are only valid in the search field if the search is applied to windows with text documents.
