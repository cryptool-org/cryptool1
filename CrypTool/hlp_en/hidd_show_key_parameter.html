Dialog Public RSA parameters

This help is accessed in 2 different ways:

  1. One way is to select the Display generated key pair option in the Generation of asymmetric key pair dialog and then click the Generate new key pair button. The public and private keys are generated using pseudorandom numbers.

    2. Thereupon the dialog "Public Parameters" is opened.

    The currently displayed key pair is saved by clicking on Apply.

  2. Secondly you can choose a key in the Available Asymmetric Key Pairs dialog box. The dialog box "Public parameters of..." is opened if you push the Show public parameters button. By pushing F1 you gain access to this help page.
