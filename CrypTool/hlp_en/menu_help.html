Commands of the top-level menu Help

The menu Help contains the following options, which provide you with support for the application:

Starting page Displays the CrypTool starting page.
Index Displays the CrypTool help index.
Scenarios (Tutorials) Displays a list of tutorials/examples and how to work through them.
Readme Displays the readme file for the CrypTool project.
CT Book Opens the CT Book as PDF document.
Presentation Opens the CrypTool presentation as PDF document.
About CrypTool Displays the version number and copyright notice for CrypTool.
