Print Preview toolbar

The Print Preview toolbar offers you the following options:

Print

Clicking this button takes you to the Print dialogue box, in which you can start a print job.

Next page

Displays the next printed page.

Previous page

Displays the previous printed page.

One Page/Two Pages

Determines whether one or two pages should be visible at the same time.

Magnifier

This option enables you to take a closer look at the page.

Zoom Out

Enabkes you to view more of the printed page.

Close

Takes you back from the Print Preview to the editing window.
