Survey for the future development of CrypTool

CrypTool is currently developed under Visual C++ v6 with the Microsoft Foundation Classes (MFC). This was a good choice when the project started in 1996. For a forward-looking and modern future development, we plan to build on other development platforms.

Currently we plan to port CrypTool to Java or C#. We would like to ask people interested in CrypTool to tell us their opinion about this matter.

Visual C++ / MFC (as is)
Java / Swing
Java / SWT (with Eclipse)
C# / .NET / Windows Forms
C++ v7 / .NET / Windows Forms

  • Your reasons
  • Are you willing to help during the development?
