    Cryptographic Hashes

    To check if your downloaded files are unmodified or have been tampered with, you can calculate their hashes and compare them with the published hashes on this web page.

    To calculate hashes you can use tools like md5sum.exe or sha1sum.exe or use CrypTool itself: menu "Indiv. Procedures / Hash / Hash Value of a File...", select SHA-1 or MD5 method.

    The cryptographic hash values of all published CrypTool versions can be found here: SHA-1, MD5.
