Number Shark manual

At the beginning the shell square will be filled with a certain amount of numbers (for example the numbers 1-20).

1. The game idea

The game idea of the Number Shark game is to win as many of the numbers as possible for yourself, so that the sum of your numbers is higher than the sum of the Number Shark, and also to reach the highest possible score. When you choose a number, the Number Shark gets all free real factors etc. (the real factors of the number "6" are "1", "2", and "3").

The Number Shark game playfully teaches the handling with factors and prime numbers and was originally designed for school kids.

In the following paragraph the handling of the Number Shark program is described.

2. Handling of the game - Main dialog

The game starts once you picked a number -- the Number Shark gets the factors of this number. This number and its factors will be deactivated in the shell square, you can’t use them any longer.

Once you and the Number Shark completed one round the next steps are:

You take a number and the Number Shark gets the factors - until all numbers are used.

Your goal is that the sum of your numbers is higher than the sum of the numbers the Number Shark eats.

In the group box "Settings for a new game" you can select with how many numbers you want to play the game. This decision needs to be confirmed by the OK button or the Enter key on your keyboard. When you first start the game this field contains the value 20.
If you press New game the number inserted lastly will be conserved. In the "Game Options" dialog you can decide if the program should remember this number for the next start too.

If you press the OK button, the number store will be filled with shells which you can use as normal buttons. You only can see 30 shells at once in the block. If you choose a number higher than 30, further tabs will appear above the block via which you can see all other shells.

With the tabulator and the arrow keys on your keyboard, you can move through the numbers and select them with the Enter key. For a better readability, all numbers are labeled in yellow. At the beginning of the game, all shells are blue, which means, that you can select anyone of it.

Every number you chose, and every factor of this number, will be colored in gray to show you that you cannot use this number any longer.

The game progress list which appears instead of the shark pictures after your first choice shows you the current score. The list is divided into six columns.

  • the current round,
  • the number you have chosen in this round,
  • which numbers the Number Shark had taken,
  • your current points,
  • the points of the Number Shark and
  • how many numbers are left in the game.
The width of the columns in the game progress list can be changed by the player, but they will not be adjusted automatically. Each row in the list can be selected and if the list is full a scroll bar will appear so that you can always scroll to every round.

Always when a prime number was chosen, no matter from whom, "(prime)" will be written behind this number in the game progress list to show that this number is a prime number.

The following buttons will help you to control the game.

When you press the button New game the course of the game and the shells are set back to the status they have at the beginning of the game. Only the amount numbers with which you played the last game will be kept.


The button Shark food only gets active if there are numbers left in the game which have no more real factors and at the same time are not a factor of any other number.
If you press this button, all these numbers will be given to the Number Shark, because sooner or later he definitely gets these numbers.


With the More options button you get access to the options screen, which you also saw at the beginning of the game.


With Undo you can repeat previous rounds if you want to choose other numbers. With Redo you can go forward through the game as many steps as you went back via Undo.

After you pressed the first shell or selected the first number you can see an LED-like display at the bottom of the dialog. On the left side you can see you current points and on the right side the points from the Number Shark.

While playing the game you can alternatingly make the shells on the buttons visible and invisible by pressing SHIFT+ALT+C on your keyboard

3. Game rules

Via the Help button you can take a look at the online help of the Number Shark game. This is also possible via the F1 key on your keyboard.


Via the button Rules of the game you get a short summary of the game rules.

4. Setting the game options

You can reach the options by pressing the More options button in the main dialog.

4.1: On the Welcome tab you can read some information about the Number Shark game.

4.2: In the Standard tab you have the following possibilities:

  • You can insert your player name, which will be written into the Summary.txt at the end of the game and it is shown in the head of the main dialog.
  • If the checkbox "Activate tool tips" is set, then the game will show you all factors of a number. This should only be used by beginners who don’t have enough experience and know-how with this game, factors and prime numbers.
  • With "Show dialog at next game start" you can decide if you want to see the options dialog at the next game start or not.
  • With the checkbox "Remember the last total number..." the game stores this number for the next game start.

All these settings can be changed in the game at any time.

4.3: Via the tab Maximum score the program can show you how many points you can get with your chosen count of numbers.

The calculation of these optimal values with our brute-force algorithm takes very long even for numbers up to 20. We believe a search for n=23 will take around 80 days on a 3 GHz PC. But even the new backtracking algorithm needs over ten minutes for numbers bigger than 40.

Therefore the results for n=158 have been pre-calculated. If the radio button Show highest score possible is checked, the result will be displayed immediately after clicking the Start button.

Beside the maximum possible points, you also get one way on which you can achieve this score, which means that the program shows you in which order you should take the numbers. Sometimes the order can be changed in some parts and you will also achieve the highest score (there is no unique solution). And you get a hint with which number you need to start the game in order to achieve the highest score.

4.4: Below the tab Accelerators you can find the implemented keyboard shortcuts.
